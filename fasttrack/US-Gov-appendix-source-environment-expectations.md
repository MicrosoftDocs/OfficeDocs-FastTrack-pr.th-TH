# <a name="source-environment-expectations-for-office-365-us-government"></a>ความคาดหวังสภาพแวดล้อมแหล่งที่มาสำหรับสำนักงาน๓๖๕รัฐบาลสหรัฐอเมริกา

สวัสดิการศูนย์ FastTrack ให้คำแนะนำสำหรับคุณในการตั้งค่าระดับการรวมกับสภาพแวดล้อมต้นทางของคุณ (ตัวอย่างเช่นถ้าคุณมีบริการในสภาพแวดล้อมต้นทางของคุณที่คุณต้องการย้ายไปยัง Office ๓๖๕)
  
> [!NOTE]
> สภาพแวดล้อมต้นทางของคุณต้องอยู่ในระดับต่ำสุดสำหรับโปรแกรมประยุกต์นั้น 
  
ตารางต่อไปนี้แสดงความคาดหวังสำหรับสภาพแวดล้อมต้นทางที่มีอยู่ของคุณสำหรับปฐมนิเทศ

|**กิจกรรม**|**สภาพแวดล้อมของแหล่งที่มาคาดหวัง**|
|:-----|:-----|
|คอร์ปฐมนิเทศ | ฟอเรสต์ที่ใช้งานอยู่ไดเรกทอรีที่มีการตั้งค่าระดับป่าไม้ทำการที่ Windows Server ๒๐๐๓เป็นต้นไปด้วยการกำหนดค่าฟอเรสต์ต่อไปนี้:  <br/>      ฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่เดียว  <br/>    ที่ใช้งานอยู่บัญชีไดเรกทอรีเดียวฟอเรสต์และทรัพยากรฟอเรสต์ (Exchange และ/หรือ Lync ๒๐๑๐, Lync ๒๐๑๓หรือ Skype สำหรับธุรกิจ) โท  <br/>  หลายบัญชีของไดเรกทอรีที่ใช้งานอยู่และฟอเรสต์ทรัพยากร (อัตราแลกเปลี่ยนและ/หรือ Lync ๒๐๑๐, Lync ๒๐๑๓หรือ Skype for Business)  <br/>  หลายบัญชีไดเรกทอรีที่ใช้งานอยู่ที่มีฟอเรสต์เป็นหนึ่งในฟอเรสต์บัญชีผู้ใช้ Active Directory ที่มี Exchange และ/หรือ Lync ๒๐๑๐, Lync ๒๐๑๓หรือ Skype สำหรับธุรกิจ  <br/>  หลายบัญชีไดเรกทอรีที่ใช้งานอยู่ฟอเรสต์แต่ละมีองค์กรแลกเปลี่ยนของตนเอง <br/> <br/> **หมายเหตุ**  *สำหรับสถานการณ์ที่ใช้งานอยู่หลายฟอเรสต์ไดเรกทอรีถ้า Lync ๒๐๑๐, Lync ๒๐๑๓หรือ Skype สำหรับธุรกิจถูกจัดวางจะต้องถูกปรับใช้ในฟอเรสต์ active directory เดียวกันเป็นอัตราแลกเปลี่ยน*   <br/>  <br/>      **หมายเหตุ***เมื่อใช้งานหลาย Forests ไดเรกทอรีที่มีหลายองค์กรแลกเปลี่ยนในการกำหนดค่าการแลกเปลี่ยนหลายไฮบริดสลีชื่อหลัก (UPN) ของผู้ใช้ที่ใช้ร่วมกันระหว่าง forests ที่มาไม่ได้รับการสนับสนุน คุณควรจะแยก SMTP namespaces หลักระหว่างองค์กรแลกเปลี่ยน สำหรับข้อมูลเพิ่มเติมให้ดูการปรับใช้[ไฮบริดสลีกับฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่หลาย](https://go.microsoft.com/fwlink/?linkid=845444)*       <br/>   <br/>   **หมายเหตุ**  *สำหรับการกำหนดค่าทั้งหมด forests การปรับใช้ fs โฆษณาอยู่นอกขอบเขตสำหรับสวัสดิการศูนย์ FastTrack*           |
|การแลกเปลี่ยนแบบออนไลน์ปฐมนิเทศ | สภาพแวดล้อมของคุณต้องมีหนึ่งในระดับต่ำสุดดังต่อไปนี้:  <br/>  องค์กรแลกเปลี่ยนเดียวหรือหลายที่มีเซิร์ฟเวอร์ Exchange ๒๐๐๓เป็นต้นไป  <br/>  เดียว IBM Domino 7.0.3 เป็นต้นไปสภาพแวดล้อม ([ภาคผนวก A-โยกย้ายจาก IBM Domino แลกเปลี่ยนแบบออนไลน์](O365-from-ibm-domino-to-exchange-online.md)).  <br/>  สภาพแวดล้อมอีเมลการเข้าถึงข้อความอินเทอร์เน็ตเดียว (IMAP)-ที่สามารถ  <br/>  สภาพแวดล้อมแบบ G Suite (Gmail, รายชื่อและปฏิทินเท่านั้น)  <br/>  GroupWise เดียวของ Novell 7.0.4 สภาพแวดล้อมที่เป็นไปได้            |
|SharePoint แบบออนไลน์และ OneDrive สำหรับธุรกิจปฐมนิเทศ  | **SharePoint ไฮบริดสลี**  <br/>  การกำหนดค่า SharePoint ไฮบริดสลีรวมถึงการตั้งค่าคอนฟิกการค้นหาไฮบริ, ไซต์, ระบบภาษี, ชนิดเนื้อหา, OneDrive สำหรับธุรกิจ, เปิดใช้งานโปรแกรมประยุกต์ขยาย, ไซต์เอ็กซ์ทราเน็ตและการสร้างไซต์ด้วยตนเองที่เชื่อมต่อจากในสถานที่ไปยัง SharePoint เป้าหมายเดียว สภาพแวดล้อมออนไลน์ เมื่อต้องการเปิดใช้งาน SharePoint ไฮบริดสลีคุณต้องมีหนึ่งในสภาพแวดล้อมของ SharePoint Server ในสถานที่ต่อไปนี้:  <br/> <br/> ***SharePoint Server ๒๐๑๓***  <br/>  มิถุนายน๒๐๑๗ปรับปรุงสาธารณะ (PU) สำหรับชนิดเนื้อหาไฮบริดสลี สำหรับข้อมูลเพิ่มเติมให้ดูที่การ[ตั้งค่าคอนฟิกระบบภาษี SharePoint แบบผสมและชนิดเนื้อหาไฮบริดสลี](https://go.microsoft.com/fwlink/?linkid=853547)  <br/>  มีนาคม๒๐๑๗ PU สำหรับการกำหนดค่าการสร้างไซต์ด้วยตนเอง การสร้างไซต์แบบไฮบริดสลีด้วยตนเองเท่านั้นสนับสนุนกับ SharePoint Server ๒๐๑๓ สำหรับข้อมูลเพิ่มเติมให้ดู[การสร้างไซต์แบบไฮบริดสลีด้วยตนเอง](https://go.microsoft.com/fwlink/?linkid=846015)  <br/>  พฤศจิกายน๒๐๑๖ PU เป็นต้นไปด้วยระบบภาษีไฮบริดสลี สำหรับข้อมูลเพิ่มเติมให้ดูที่การ[วางแผนระบบภาษี SharePoint แบบไฮบริดสลี](https://go.microsoft.com/fwlink/?linkid=844867)  <br/>  กรกฎาคม๒๐๑๖ PU เป็นต้นไปสำหรับสถานการณ์การกำหนดค่าไฮบริดสลีอื่นๆทั้งหมด  <br/><br/> **หมายเหตุ:**  *สถานการณ์ไฮบริดสลี๒๐๑๓เซิร์ฟเวอร์ sharepoint ได้รับการสนับสนุนกับ SharePoint Server ๒๐๑๓เท่านั้น สถานการณ์เหล่านี้ไม่ได้รับการสนับสนุนใน SharePoint Foundation ๒๐๑๓*  <br/>   <br/>    ***SharePoint Server ๒๐๑๖***  <br/>  มิถุนายน๒๐๑๗ PU สำหรับชนิดของเนื้อหาไฮบริดสลี สำหรับข้อมูลเพิ่มเติมให้ดูที่การ[ตั้งค่าคอนฟิกระบบภาษี SharePoint แบบผสมและชนิดเนื้อหาไฮบริดสลี](https://go.microsoft.com/fwlink/?linkid=853547)  <br/>  พฤศจิกายน๒๐๑๖ PU (คุณลักษณะ Pack 1) ที่มีระบบภาษีไฮบริดสลี สำหรับข้อมูลเพิ่มเติมให้ดูที่การ[วางแผนระบบภาษี **หมายเหตุ:***การปรับรุ่นของสภาพแวดล้อม sharepoint ในสถานที่ไปยังเซิร์ฟเวอร์ SharePoint ๒๐๑๓หรือ sharepoint server ๒๐๑๖ได้รับผลประโยชน์จากศูนย์ FastTrack สำหรับข้อมูลเพิ่มเติมให้ดู[ระดับการปรับปรุงสาธารณะขั้นต่ำสำหรับลักษณะการทำงานของ SharePoint ไฮบริดสลี](https://go.microsoft.com/fwlink/?linkid=853548)*               |
|Skype สำหรับธุรกิจออนไลน์ปฐมนิเทศ  | **การประเมินเครือข่าย**  <br/>  การตรวจสอบพอร์ตและปลายทาง  <br/>  การตรวจสอบคุณภาพการเชื่อมต่อ  <br/>  การประมาณแบนด์วิดธ์  <br/><br/>  **ไฮบริดสลี Lync**  <br/>  ฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่ในสถานเดียว  <br/>  สภาพแวดล้อมแบบ Lync ๒๐๑๐ Server กับ Lync ๒๐๑๓ admin เครื่องมือหรือ Skype สำหรับธุรกิจ๒๐๑๕ผู้ดูแลเครื่องมือและบทบาทของเซิร์ฟเวอร์ Lync ๒๐๑๐ขอบ Server  <br/>  สภาพแวดล้อมแบบ Lync ๒๐๑๓ Server และบทบาทของเซิร์ฟเวอร์ Lync ๒๐๑๓ขอบ  <br/><br/>  **Skype สำหรับไฮบริดสลีธุรกิจออนไลน์**  <br/>  ฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่ในสถานเดียว  <br/>  ฟอเรสต์บัญชีผู้ใช้ Active Directory เดียวเป็นต้นไปและฟอเรสต์ทรัพยากร (Exchange และ/หรือ Skype สำหรับธุรกิจ) โท  <br/>  หลายบัญชีไดเรกทอรีที่ใช้งานอยู่ฟอเรสต์กับฟอเรสต์หนึ่งเป็นแบบรวมศูนย์รวมบัญชีผู้ใช้ที่มีการแลกเปลี่ยนและ/หรือ Skype สำหรับธุรกิจในนั้น  <br/>  การ Skype สำหรับสภาพแวดล้อม๒๐๑๕เซิร์ฟเวอร์ธุรกิจรวมทั้ง Skype สำหรับบทบาทเซิร์ฟเวอร์ขอบธุรกิจ  <br/><br/> **หมายเหตุ:**  *บริการเพิ่มเติมนี้มีไว้สำหรับการตั้งค่าคอนฟิกและการตรวจสอบของงานแยกโดเมน (ไฮบริดสลี) และไม่รวมถึงการแนะนำคอมโพเนนต์ในสถานที่ (ตัวอย่างเช่น Lync ๒๐๑๓ admin เครื่องมือหรือ Lync 2013/Skype สำหรับเซิร์ฟเวอร์ธุรกิจออนไลน์หรือ Lync ๒๐๑๐, lync ๒๐๑๓*    <br/><br/>        **อุปกรณ์ห้องประชุม**  <br/>  การสร้างบัญชีออนไลน์ที่จำเป็นสำหรับอุปกรณ์ห้องประชุมที่ได้รับการสนับสนุนที่ระบุไว้ในการระบบห้องประชุมแท็บในการ[lync สำหรับแค็ตตาล็อกโซลูชันทางธุรกิจ](https://go.microsoft.com/fwlink/?LinkId=615775)  <br/><br/>  **การเปิดใช้งานการประชุมทางเสียง**  <br/>  การตั้งค่าองค์กรสำหรับการประชุมที่เป็นค่าเริ่มต้นของบริดจ์  <br/>  การกำหนดสะพานการประชุมให้กับผู้ใช้ที่ได้รับอนุญาต  <br/><br/>  **การเปิดใช้งานระบบโทรศัพท์และคำแนะนำแผนการโทร (เฉพาะเรา–ไม่มีให้บริการใน GCC แผนสูงหรือ DoD)**  <br/>  การตั้งค่าองค์กรสำหรับระบบคลาวด์การติดตั้งเริ่มต้น  <br/>  การกำหนดหมายเลขให้กับผู้ใช้ที่ได้รับอนุญาต  <br/>  หมายเลขท้องถิ่น porting คำแนะนำผ่านส่วนติดต่อผู้ใช้ (UI) ถึง๙๙๙  <br/>  การสนับสนุนหมายเลขท้องถิ่น porting การร้องขอการบริการ (SR) มากกว่า๙๙๙
