# <a name="source-environment-expectations-for-office-365-us-government"></a>ความคาดหวังด้านสภาพแวดล้อมแหล่งที่มาสำหรับ Office ๓๖๕รัฐบาลสหรัฐฯ

ประโยชน์ของศูนย์ FastTrack ให้คำแนะนำสำหรับคุณในการตั้งค่าระดับของการรวมกับสภาพแวดล้อมต้นทางของคุณ (ตัวอย่างเช่นถ้าคุณมีบริการในสภาพแวดล้อมของแหล่งที่มาของคุณที่คุณต้องการย้ายไปยัง Office ๓๖๕)
  
> [!NOTE]
> ถ้าจำเป็นต้องมีการรวมสภาพแวดล้อมต้นทางของคุณต้องอยู่ในระดับต่ำสุดสำหรับโปรแกรมประยุกต์นั้น 
  
ตารางต่อไปนี้แสดงความคาดหวังสำหรับสภาพแวดล้อมต้นทางที่มีอยู่ของคุณสำหรับการเตรียมการ

|**กิจกรรม**|**คาดการณ์สภาพแวดล้อมต้นทาง**|
|:-----|:-----|
|การเตรียมการหลัก | ฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่ที่มีการตั้งค่าระดับฟอเรสต์ทำหน้าที่เป็น Windows Server ๒๐๐๓เป็นต้นไปโดยมีการกำหนดค่าฟอเรสต์ต่อไปนี้:  <br/>      ฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่เดียว  <br/>    ฟอเรสต์บัญชีที่ใช้งานอยู่เดียวและฟอเรสต์ทรัพยากร (Exchange และ/หรือ Lync ๒๐๑๐, Lync ๒๐๑๓หรือ Skype สำหรับธุรกิจ) โทโพโลยี  <br/>  หลายบัญชีไดเรกทอรีที่ใช้งานอยู่ป่าไม้และทรัพยากรฟอเรสต์ (แลกเปลี่ยนและ/หรือ Lync ๒๐๑๐, Lync ๒๐๑๓หรือ Skype สำหรับธุรกิจ) โทโพโลยี  <br/>  หลายบัญชีไดเรกทอรีที่ใช้งานอยู่ป่าไม้กับหนึ่งในฟอเรสต์เป็นส่วนกลางของบัญชีผู้ใช้ Active Directory ฟอเรสต์ที่รวมถึงการแลกเปลี่ยนและ/หรือ Lync ๒๐๑๐, Lync ๒๐๑๓หรือ Skype สำหรับธุรกิจ  <br/>  หลายบัญชีไดเรกทอรีที่ใช้งานอยู่ของฟอเรสต์แต่ละคนมีองค์กรแลกเปลี่ยนของตัวเอง <br/> <br/> **ทราบ** *สำหรับสถานการณ์จำลองไดเรกทอรีที่ใช้งานอยู่หลายฟอเรสต์ถ้า lync ๒๐๑๐, lync ๒๐๑๓หรือ Skype สำหรับธุรกิจถูกปรับใช้จะต้องถูกนำไปใช้ในฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่เดียวกับการแลกเปลี่ยน*     <br/>  <br/>      **ทราบ** *เมื่อมีการนำไปใช้หลายป่าไม้ของไดเรกทอรีที่มีหลายองค์กร exchange ในการตั้งค่าคอนฟิกไฮบริดสลีแลกเปลี่ยนหลาย, ชื่อผู้ใช้หลัก (UPN) namespace ระหว่างฟอเรสต์ต้นทางไม่ได้รับการสนับสนุน Namespace SMTP หลักระหว่างองค์กร Exchange ควรจะแยกต่างหาก สำหรับข้อมูลเพิ่มเติมให้ดูการปรับใช้[ไฮบริดสลีกับหลายฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่](https://go.microsoft.com/fwlink/?linkid=845444)*       <br/>   <br/>   **ทราบ** *สำหรับการตั้งค่าคอนฟิกหลายฟอเรสต์ทั้งหมดการปรับใช้ FS โฆษณาอยู่นอกขอบเขตสำหรับสวัสดิการศูนย์ FastTrack*             |
|การเปลี่ยนสถานะการออนไลน์ของ Exchange | สภาพแวดล้อมของคุณต้องมีหนึ่งในระดับต่ำสุดต่อไปนี้:  <br/>  องค์กรแลกเปลี่ยนเดียวหรือหลายหน่วยกับ Exchange Server ๒๐๐๓เป็นต้นไป  <br/>  หนึ่ง IBM Domino 7.0.3 เป็นต้นไปสภาพแวดล้อม ([ภาคผนวก A-การโยกย้ายจาก IBM Domino เพื่อแลกเปลี่ยนแบบออนไลน์](O365-from-ibm-domino-to-exchange-online.md))  <br/>  สภาพแวดล้อมแบบเมลที่สามารถเข้าถึงข้อความอินเทอร์เน็ต (IMAP) แบบเดียว  <br/>  สภาพแวดล้อม G Suite เดียว (Gmail ผู้ติดต่อและปฏิทินเท่านั้น)  <br/>  เดียว Novell GroupWise 7.0.4 สภาพแวดล้อมที่เป็นไปได้            |
|SharePoint แบบออนไลน์และ OneDrive สำหรับการเตรียมธุรกิจ  | **ไฮบริดสลี SharePoint**  <br/>  การกำหนดค่าไฮบริดสลี SharePoint รวมถึงการกำหนดค่าการค้นหาไฮบริดสลีไซต์ระบบภาษีชนิดเนื้อหา OneDrive สำหรับธุรกิจตัวเปิดใช้โปรแกรมประยุกต์แบบขยายเว็บไซต์เอกซ์ทราเน็ตและการสร้างไซต์บริการตนเองที่เชื่อมต่อจากในสถานที่ไปยัง SharePoint เป้าหมายเดียว สภาพแวดล้อมออนไลน์ เมื่อต้องการเปิดใช้งานไฮบริดสลี SharePoint คุณต้องมีหนึ่งในสภาพแวดล้อมของ SharePoint Server ในสถานต่อไปนี้:  <br/> <br/> ***เซิร์ฟเวอร์ SharePoint ๒๐๑๓***  <br/>  มิถุนายน๒๐๑๗สาธารณะปรับปรุง (PU) สำหรับชนิดเนื้อหาไฮบริดสลี สำหรับข้อมูลเพิ่มเติมโปรดดูที่การ[ตั้งค่าคอนฟิกระบบภาษี SharePoint ไฮบริดสลีและชนิดเนื้อหาไฮบริดสลี](https://go.microsoft.com/fwlink/?linkid=853547)  <br/>  มีนาคม๒๐๑๗ PU สำหรับการกำหนดค่าการสร้างไซต์ด้วยตนเอง การสร้างไซต์บริการตนเองแบบไฮบริดสลีได้รับการสนับสนุนเฉพาะกับ SharePoint Server ๒๐๑๓เท่านั้น สำหรับข้อมูลเพิ่มเติมให้ดู[การสร้างไซต์แบบบริการตนเองของไฮบริดสลี](https://go.microsoft.com/fwlink/?linkid=846015)  <br/>  พฤศจิกายน๒๐๑๖ PU เป็นไปได้กับระบบภาษีไฮบริดสลี สำหรับข้อมูลเพิ่มเติมโปรดดูที่[ระบบภาษี SharePoint ไฮบริดสลีการวางแผน](https://go.microsoft.com/fwlink/?linkid=844867)  <br/>  กรกฎาคม๒๐๑๖ PU เป็นต้นไปสำหรับสถานการณ์จำลองการตั้งค่าคอนฟิกไฮบริดสลีอื่นๆทั้งหมด  <br/><br/> **ทราบ** *สถานการณ์จำลองไฮบริดสลี๒๐๑๓เซิร์ฟเวอร์ sharepoint ได้รับการสนับสนุนเฉพาะกับ Sharepoint Server ๒๐๑๓เท่านั้น สถานการณ์เหล่านี้ไม่ได้รับการสนับสนุนใน SharePoint Foundation ๒๐๑๓*    <br/>   <br/>    ***เซิร์ฟเวอร์ SharePoint ๒๐๑๖***  <br/>  มิถุนายน๒๐๑๗ PU สำหรับชนิดเนื้อหาไฮบริดสลี สำหรับข้อมูลเพิ่มเติมโปรดดูที่การ[ตั้งค่าคอนฟิกระบบภาษี SharePoint ไฮบริดสลีและชนิดเนื้อหาไฮบริดสลี](https://go.microsoft.com/fwlink/?linkid=853547)  <br/>  พฤศจิกายน๒๐๑๖ PU (คุณลักษณะ Pack 1) ที่มีระบบภาษีไฮบริดสลี สำหรับข้อมูลเพิ่มเติมโปรดดูที่[ระบบภาษี SharePoint ไฮบริดสลีการวางแผน](https://go.microsoft.com/fwlink/?linkid=844867)  <br/>  กรกฎาคม๒๐๑๖ PU เป็นต้นไปสำหรับสถานการณ์จำลองการตั้งค่าคอนฟิกไฮบริดสลีอื่นๆทั้งหมด  <br/><br/> **ทราบ** การ*ปรับรุ่นของสภาพแวดล้อม sharepoint ในสถานที่ไปยังเซิร์ฟเวอร์ sharepoint ๒๐๑๓หรือ Sharepoint server ๒๐๑๖ไม่ได้จัดเตรียมโดยสิทธิประโยชน์ของศูนย์ FastTrack สำหรับข้อมูลเพิ่มเติมให้ดู[ระดับการปรับปรุงสาธารณะขั้นต่ำสำหรับคุณลักษณะไฮบริดสลี SharePoint](https://go.microsoft.com/fwlink/?linkid=853548)*               |
|Skype สำหรับการปฐมนิเทศธุรกิจแบบออนไลน์  | **การประเมินเครือข่าย**  <br/>  การตรวจสอบพอร์ตและปลายทาง  <br/>  การตรวจสอบคุณภาพการเชื่อมต่อ  <br/>  การประมาณแบนด์วิดท์  <br/><br/>  **ไฮบริดสลี Lync**  <br/>  ฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่ในสถานที่เดียว  <br/>  สภาพแวดล้อมเซิร์ฟเวอร์ Lync ๒๐๑๐กับเครื่องมือการดูแลระบบ Lync ๒๐๑๓หรือ Skype สำหรับธุรกิจ๒๐๑๕เครื่องมือผู้ดูแลระบบและบทบาทบนเซิร์ฟเวอร์ Lync ๒๐๑๐  <br/>  สภาพแวดล้อมเซิร์ฟเวอร์ Lync ๒๐๑๓และบทบาทเซิร์ฟเวอร์ขอบ Lync ๒๐๑๓  <br/><br/>  **Skype สำหรับไฮบริดสลีธุรกิจออนไลน์**  <br/>  ฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่ในสถานที่เดียว  <br/>  ฟอเรสต์บัญชีไดเรกทอรีที่ใช้งานอยู่เดียวเป็นต้นและทรัพยากรฟอเรสต์ (แลกเปลี่ยนและ/หรือ Skype สำหรับธุรกิจ) โทโพโลยี  <br/>  หลายบัญชีไดเรกทอรีที่ใช้งานอยู่ฟอเรสต์กับฟอเรสต์หนึ่งเป็นส่วนกลางของบัญชีไดเรกทอรีที่ใช้งานอยู่ป่ากับการแลกเปลี่ยนและ/หรือ Skype สำหรับธุรกิจในนั้น  <br/>  แบบ Skype สำหรับสภาพแวดล้อมทางธุรกิจเซิร์ฟเวอร์๒๐๑๕รวมทั้ง Skype สำหรับบทบาทเซิร์ฟเวอร์ขอบธุรกิจ  <br/><br/> **ทราบ** *บริการเพิ่มเติมนี้มีไว้สำหรับการตั้งค่าคอนฟิกและการตรวจสอบงานแยกโดเมน (ไฮบริดสลี) และไม่รวมถึงการแนะนำคอมโพเนนต์ในสถานที่ (ตัวอย่างเช่น lync ๒๐๑๓ admin เครื่องมือหรือ lync 2013/Skype สำหรับเซิร์ฟเวอร์ธุรกิจออนไลน์หรือ lync ๒๐๑๐, lync ๒๐๑๓หรือ Skype สำหรับเซิร์ฟเวอร์ขอบธุรกิจ)*      <br/><br/>        **อุปกรณ์ห้องประชุม**  <br/>  การสร้างบัญชีออนไลน์ที่จำเป็นสำหรับอุปกรณ์ในห้องประชุมที่ได้รับการสนับสนุนที่แสดงอยู่บนแท็บระบบห้องประชุมในแบบ[Skype สำหรับแค็ตตาล็อกโซลูชันธุรกิจ](https://go.microsoft.com/fwlink/?LinkId=615775)  <br/><br/>  **การเปิดใช้งานการประชุมผ่านเสียง**  <br/>  การตั้งค่าองค์กรสำหรับการตั้งค่าเริ่มต้นของบริดจ์การประชุม  <br/>  การกำหนดสะพานการประชุมให้กับผู้ใช้ที่ได้รับอนุญาต  <br/><br/>  **การเปิดใช้งานระบบโทรศัพท์และการแนะนำแผนการโทร (เฉพาะสหรัฐอเมริกา–ไม่มีในแผนสูง GCC หรือ DoD)**  <br/>  การตั้งค่าองค์กรสำหรับการตั้งค่าเริ่มต้นของเสียง Cloud  <br/>  การกำหนดหมายเลขให้กับผู้ใช้ที่ได้รับอนุญาต  <br/>  หมายเลขท้องถิ่นการเพิ่มการแนะนำผ่านอินเทอร์เฟซผู้ใช้ (UI) ถึง๙๙๙  <br/>  การสนับสนุนการร้องขอการบริการของหมายเลขภายในเครื่อง (SR) มากกว่า๙๙๙  <br/><br/>  **การเปิดใช้งาน Skype สำหรับการประชุมทางธุรกิจการออกอากาศคำแนะนำการปฐมนิเทศ (ไม่มีในการแผนสูง GCC หรือ DoD)**  <br/>  การตั้งค่าองค์กรสำหรับสหพันธรัฐที่มีบริการออกอากาศการประชุม   |
