# <a name="eligibility"></a>สิทธิ์

ในหัวข้อนี้เราจะจัดเค้าร่างข้อกำหนดสิทธิ์สำหรับการรับระยะไกลความช่วยเหลือที่แนะนำจากผู้เชี่ยวชาญ FastTrack รวมอยู่ในการซื้อสิทธิ์การใช้งาน Microsoft ๓๖๕และรายละเอียดผลิตภัณฑ์และความสามารถที่เฉพาะเจาะจงที่ได้รับการคุ้มครอง

การช่วยเหลือ FastTrack พร้อมใช้งานสำหรับลูกค้าที่มี๑๕๐หรือสิทธิ์การใช้งานเพิ่มเติมจากหนึ่งในแผนการใช้งานที่มีสิทธิ์จาก Microsoft ตระกูลผลิตภัณฑ์ต่อไปนี้: Microsoft ๓๖๕, Office ๓๖๕, Enterprise Mobility & ความปลอดภัยและ Windows 10 แผนเหล่านี้สามารถใช้ได้กับผลิตภัณฑ์แต่ละรายการ (เช่น Exchange Online) หรือชุดผลิตภัณฑ์ (Office ๓๖๕ E3) เนื่องจาก Microsoft มีตัวเลือกการจัดซื้อหลายรายการรายการของแผนที่มีสิทธิ์จะครอบคลุมและมีอยู่ในแผนที่มี[สิทธิ์](#eligible-plans)

## <a name="products-and-capabilities-covered"></a>ผลิตภัณฑ์และความสามารถที่ครอบคลุม

ผู้เชี่ยวชาญ FastTrack สามารถให้ความช่วยเหลือที่แนะนำสำหรับบริการต่อไปนี้ซึ่งอาจรวมอยู่ในแผนการซื้อของคุณ:

  - Azure Active Directory (Azure AD) Premium
  - การป้องกันข้อมูล Azure
  - Exchange Online
  - แอป Microsoft ๓๖๕

> Office ๓๖๕ ProPlus ได้รับการเปลี่ยนชื่อเป็นแอป Microsoft ๓๖๕ ในบางกรณีคุณอาจพบว่าการใช้ชื่อเดิม สำหรับรายละเอียดของการเปลี่ยนแปลงนี้จะมีผลโปรดดูที่[การเปลี่ยนแปลงชื่อสำหรับ Office ๓๖๕ ProPlus](https://docs.microsoft.com/deployoffice/name-change)

  - การป้องกันภัยคุกคามขั้นสูงของ Microsoft Defender (ATP)
  - Microsoft Edge
  - การกำกับดูแลข้อมูลของ Microsoft
  - การป้องกันข้อมูลของ Microsoft
  - Microsoft Intune
  - Microsoft Teams
  - การป้องกันการคุกคามขั้นสูงของ Office ๓๖๕ (ATP)
  - OneDrive for Business
  - Power BI
  - Project Online
  - SharePoint Online
  - Skype for Business Online
  - Windows 10
  - Yammer Enterprise
