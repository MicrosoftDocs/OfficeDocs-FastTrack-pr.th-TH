# <a name="source-environment-expectations"></a>ข้อกำหนดของสภาพแวดล้อมต้นทาง
> [!CAUTION]
> เนื้อหานี้ไม่ได้อยู่ในปัจจุบันอีกต่อไปและมีการจัดกำหนดการสำหรับการเอาออก ใช้สารบัญในการนำทางด้านซ้ายสำหรับเนื้อหาปัจจุบัน

ประโยชน์ของศูนย์ FastTrack จะให้คำแนะนำสำหรับคุณในการตั้งค่าระดับการรวมกับสภาพแวดล้อมต้นฉบับของคุณ (ตัวอย่างเช่นถ้าคุณมีบริการในสภาพแวดล้อมต้นทางของคุณที่คุณต้องการย้ายไปยัง Office ๓๖๕)
  
> [!NOTE]
> ถ้าจำเป็นต้องมีการรวมระบบสภาพแวดล้อมของแหล่งข้อมูลของคุณต้องอยู่ในระดับน้อยที่สุดสำหรับแอปพลิเคชันนั้น
