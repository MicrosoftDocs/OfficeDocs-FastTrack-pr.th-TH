# <a name="app-assure"></a>App Assure

ประโยชน์จากศูนย์ FastTrack สําหรับ Windows 10 ให้การเข้าถึง App Assure ซึ่งเป็นบริการที่ออกแบบมาเพื่อแก้ไขปัญหาเกี่ยวกับความเข้ากันได้ของแอป Windows 10 และ Microsoft 365 Apps เมื่อคุณร้องขอบริการ App Assure วิศวกรของ Microsoft จะทํางานร่วมกับคุณเพื่อจัดการกับปัญหาของแอปที่ถูกต้องโดยไม่มีค่าใช้จ่ายเพิ่มเติมกับคุณด้วยการสมัครใช้งานที่มีสิทธิ์ เรายังให้คําแนะนําแก่ลูกค้าที่ประสบปัญหาความเข้ากันได้เมื่อปรับใช้เดสก์ท็อปเสมือนของ Windows และ Microsoft Edge ใหม่ 

หากต้องการขอความช่วยเหลือ ให้กรอกแบบฟอร์ม[คําขอบริการ App Assure](https://go.microsoft.com/fwlink/?linkid=2022721)

  > [!NOTE]
> นี่คือบริการที่ FastTrack จัดหาและไม่ใช่แพคเกจซอฟต์แวร์
