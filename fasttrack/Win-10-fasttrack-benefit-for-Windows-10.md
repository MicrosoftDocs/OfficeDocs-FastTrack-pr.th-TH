# <a name="fasttrack-center-benefit-for-windows-10"></a>สิทธิประโยชน์จากศูนย์ FastTrack สำหรับ Windows 10

คุณสามารถใช้ FastTrack ศูนย์สวัสดิการสำหรับ Windows 10 เมื่อคุณซื้อสิทธิ์การใช้งานอย่างน้อย 150 สำหรับบริการที่เหมาะสมหรือแผน คุณแล้วทำงานกับผู้เชี่ยวชาญ FastTrack ที่ให้คำแนะนำคำแนะนำและด้านถ้าคุณพบปัญหาความเข้ากันได้ของโปรแกรมประยุกต์คุณจัดวาง Windows 10 และ Office 365 ProPlus และคอยติดตามสถานการณ์ปัจจุบันที่ไม่มีต้นทุนเพิ่มเติม (โดยมีสิทธิ์บอกรับเป็นสมาชิก)
  
> [!NOTE]
> วัตถุประสงค์ของเอกสารนี้ ผู้เชี่ยวชาญ FastTrack อาจรวมถึง Microsoft พนักงาน ผู้จัดจำหน่ายที่ได้รับการอนุมัติจาก Microsoft และได้รับการอนุมัติจาก Microsoft คู่ค้า
