# <a name="onboarding-and-migration-phases-for-office-365-us-government"></a>ขั้นตอนการเริ่มต้นและการย้ายถิ่นสําหรับ Office 365 รัฐบาลสหรัฐ

การเริ่มต้น Office 365 มีสี่ขั้นตอนหลัก ได้แก่ เริ่มต้น ประเมิน แก้ไข และเปิดใช้งาน คุณสามารถติดตามระยะเหล่านี้ได้ด้วยเฟสการย้ายข้อมูลที่เลือกได้ดังที่แสดงในรูปต่อไปนี้
  
![ขั้นตอนของผลประโยชน์บนเรือ](media/O365-Onboarding-Phases.png)
  
สําหรับรายละเอียดงานสําหรับแต่ละระยะ ให้ดูที่[ความรับผิดชอบของ FastTrack สําหรับ Office 365 รัฐบาลสหรัฐ](US-Gov-appendix-fasttrack-responsibilities.md)และ[ความรับผิดชอบของคุณสําหรับ Office 365 รัฐบาลสหรัฐ](US-Gov-appendix-your-responsibilities.md)
  
## <a name="initiate-phase"></a>เริ่มต้นเฟส

หลังจากที่คุณซื้อจํานวนที่เหมาะสมและชนิดของสิทธิ์การใช้งาน ให้ทําตามคําแนะนําจากอีเมลยืนยันการซื้อเพื่อเชื่อมโยงสิทธิ์การใช้งานกับผู้เช่าที่มีอยู่หรือผู้เช่าใหม่ของคุณ 
  
คุณสามารถขอความช่วยเหลือผ่าน[ศูนย์การจัดการ Microsoft 365](https://go.microsoft.com/fwlink/?linkid=2032704)หรือ[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 
 
เมื่อต้องการขอความช่วยเหลือผ่าน[ศูนย์การจัดการ Microsoft 365](https://go.microsoft.com/fwlink/?linkid=2032704)ผู้ดูแลระบบของคุณลงชื่อเข้าใช้ศูนย์การจัดการ แล้วคลิกวิดเจ็ต**ต้องการความช่วยเหลือ** 

เพื่อขอความช่วยเหลือผ่านเว็บไซต์[FastTrack](https://go.microsoft.com/fwlink/?linkid=780698): 
1.    ลงชื่อเข้าใช้[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 
2.    เลือก**ขอความช่วยเหลือสําหรับ Microsoft 365**จาก**การดําเนินการด่วน**ที่ด้านบนของหน้า Landing Page ของคุณ หรือโดยการเลือก**ขอความช่วยเหลือสําหรับ Microsoft 365**บนการ์ดการปรับใช้
3. กรอกแบบฟอร์ม**ขอความช่วยเหลือสําหรับ Microsoft 365** 

พาร์ทเนอร์ยังสามารถขอความช่วยเหลือผ่าน[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698)ในนามของลูกค้าได้ เมื่อต้องการทําเช่นนั้น:
1.    ลงชื่อเข้าใช้[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 
2.    เลือก**ลูกค้าของฉัน**
3.    ค้นหาลูกค้าของคุณหรือเลือกจากรายชื่อลูกค้าของคุณ
4.    เลือก**บริการ**
5.    กรอกแบบฟอร์ม**ขอความช่วยเหลือสําหรับ Microsoft 365**   

นอกจากนี้ คุณยังสามารถขอความช่วยเหลือจาก FastTrack Center ได้จาก[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698)ในรายการบริการที่พร้อมใช้งานสําหรับผู้เช่าของคุณ 
    
ในระหว่างขั้นตอนนี้เราหารือเกี่ยวกับกระบวนการเริ่มต้นการตรวจสอบข้อมูลของคุณและตั้งค่าการประชุม
  
ซึ่งรวมถึงการทํางานกับคุณเพื่อให้เข้าใจถึงวิธีการที่คุณตั้งใจจะใช้บริการและเป้าหมายขององค์กรและแผนจะผลักดันการใช้งานบริการ
  
![ขั้นตอนการเตรียมพร้อมระหว่างขั้นตอนเริ่มต้น](media/O365-Onboarding-Initiate.png)
  
## <a name="assess-phase"></a>ประเมินระยะ

ผู้จัดการ FastTrack ของคุณดําเนินการวางแผนความสําเร็จแบบโต้ตอบกับคุณและทีมรับเลี้ยงบุตรบุญธรรมของคุณ นี้แนะนําคุณกับความสามารถของบริการที่มีสิทธิ์ที่คุณซื้อรากฐานที่สําคัญที่คุณต้องการสําหรับความสําเร็จวิธีการสําหรับการขับขี่ของบริการและสถานการณ์ที่คุณสามารถใช้เพื่อให้ได้มูลค่าจากบริการ เราช่วยคุณในการวางแผนความสําเร็จและให้ข้อเสนอแนะเกี่ยวกับประเด็นสําคัญ (ตามความจําเป็น)
  
ผู้เชี่ยวชาญ FastTrack ทํางานร่วมกับคุณเพื่อประเมินสภาพแวดล้อมต้นทางและความต้องการของคุณ เราให้บริการเครื่องมือสําหรับคุณในการรวบรวมข้อมูลเกี่ยวกับสภาพแวดล้อมของคุณและแนะนําคุณผ่านการประเมินความต้องการแบนด์วิดธ์และการประเมินเบราว์เซอร์อินเทอร์เน็ตระบบปฏิบัติการของลูกค้าระบบชื่อโดเมน (DNS), เครือข่ายโครงสร้างพื้นฐานและระบบข้อมูลระบุตัวตนเพื่อตรวจสอบว่าการเปลี่ยนแปลงใด ๆ ที่จําเป็นสําหรับการ onboarding 
  
เราจัดเตรียมแผนแก้ไขที่นําสภาพแวดล้อมต้นทางของคุณขึ้นกับข้อกําหนดขั้นต่ําสําหรับการเริ่มต้น Office 365 ที่ประสบความสําเร็จ และถ้าจําเป็น สําหรับการโยกย้ายกล่องจดหมายและ/หรือข้อมูลสําเร็จ เราจัดเตรียมชุดกิจกรรมที่แนะนําเพื่อเพิ่มมูลค่าของผู้ใช้และการยอมรับ เรายังตั้งค่าที่เหมาะสมจุดตรวจสอบสายสําหรับเฟสแก้ไข.
  
![ขั้นตอนการเตรียมพร้อมสําหรับขั้นตอนการประเมิน](media/O365.png)
  
## <a name="remediate-phase"></a>เฟสของการแก้ไข

คุณทํางานแก้ไขตามสภาพแวดล้อมของแหล่งที่มาเพื่อให้คุณตอบสนองความต้องการสําหรับการเริ่มต้น
  
![ขั้นตอนการเตรียมพร้อมระหว่างขั้นตอนการฟื้นฟู](media/O365-Onboarding-Remediate.png)
  
นอกจากนี้เรายังมีชุดของกิจกรรมที่แนะนําเพื่อเพิ่มมูลค่าของผู้ใช้และการยอมรับ ก่อนที่จะเริ่มขั้นตอนการเปิดใช้งานเราร่วมกันตรวจสอบผลของกิจกรรมการแก้ไขเพื่อให้แน่ใจว่าคุณพร้อมที่จะดําเนินการต่อ 
  
ในระหว่างขั้นตอนนี้ FastTrack Manager ของคุณทํางานร่วมกับคุณในการวางแผนความสําเร็จแนะนําคุณไปยังทรัพยากรที่เหมาะสมและแนวทางปฏิบัติที่ดีที่สุดเพื่อให้คําแนะนําแก่คุณเพื่อให้บริการพร้อมใช้งานสําหรับองค์กรของคุณและผลักดันการใช้งานทั่วทั้งบริการ
  
## <a name="enable-phase"></a>เปิดใช้งานเฟส

เมื่อกิจกรรมการแก้ไขทั้งหมดเสร็จสมบูรณ์ โฟกัสจะเลื่อนไปกําหนดค่าโครงสร้างพื้นฐานหลักสําหรับการใช้บริการ การเตรียมใช้งาน Office 365 และดําเนินการกิจกรรมเพื่อกระตุ้นการนําบริการไปใช้ 
  
## <a name="core"></a>หลัก

การเตรียมพร้อมหลักเกี่ยวข้องกับการจัดเตรียมบริการและการรวมผู้เช่าและเอกลักษณ์ นอกจากนี้ ยังมีขั้นตอนสําหรับการให้บริการพื้นฐานสําหรับบริการออนบอร์ด เช่น Exchange Online, SharePoint Online และ Skype สําหรับธุรกิจออนไลน์ คุณและ FastTrack Manager ของคุณยังคงมีการประชุมด่านการวางแผนความสําเร็จเพื่อประเมินความคืบหน้ากับเป้าหมายของคุณและกําหนดความช่วยเหลือเพิ่มเติมที่คุณต้องการ
  
![ขั้นตอนหลักในการเริ่มใช้งานระหว่างการเปิดใช้งานphase_1](media/O365-Onboarding-Enable-Core.png)
  
![ขั้นตอนหลักในการเริ่มใช้งานระหว่างการเปิดใช้งานphase_2](media/204bbb6c-f7fe-4401-aa69-dd828ed44963.png)
  
> [!NOTE]
> WAP ย่อมาจากพร็อกซีโปรแกรมประยุกต์บนเว็บ SSL ย่อมาจากเลเยอร์ซ็อกเก็ตที่ปลอดภัย SDS ย่อมาจากการซิงค์ข้อมูลของโรงเรียน สําหรับข้อมูลเพิ่มเติมเกี่ยวกับ SDS ให้ดูที่[ยินดีต้อนรับสู่ Microsoft School Data Sync](https://go.microsoft.com/fwlink/?linkid=871480) 
  
การเตรียมพร้อมสําหรับบริการที่มีสิทธิ์หนึ่งรายการขึ้นไปสามารถเริ่มต้นเมื่อเริ่มต้นหลักเสร็จสิ้น
  
## <a name="exchange-online"></a>การแลกเปลี่ยนแบบออนไลน์

สําหรับ Exchange Online เราแนะนําให้คุณผ่านกระบวนการเพื่อให้องค์กรของคุณพร้อมที่จะใช้อีเมล ขั้นตอนที่แน่นอนขึ้นอยู่กับสภาพแวดล้อมต้นทางและแผนการโอนย้ายอีเมลของคุณอาจรวมถึงคําแนะนําสําหรับ:
  
- การตั้งค่าคุณลักษณะการป้องกันแบบออนไลน์ของ Exchange (EOP) สําหรับโดเมนทั้งหมดที่เปิดใช้งานจดหมายที่ตรวจสอบความถูกต้องใน Office 365   
    > [!NOTE]
    > ระเบียนของการแลกเปลี่ยนจดหมาย (MX) ของคุณต้องชี้ไปที่ Office 365   
- การตั้งค่าคุณลักษณะการป้องกันภัยคุกคามขั้นสูงในการแลกเปลี่ยนแบบออนไลน์ (ATP) ถ้าเป็นส่วนหนึ่งของบริการการสมัครใช้งานของคุณเมื่อระเบียน MX ของคุณชี้ไปที่ Office 365 คุณลักษณะนี้ถูกกําหนดค่าเป็นส่วนหนึ่งของการป้องกันแบบออนไลน์ของอัตราแลกเปลี่ยนการตั้งค่าการป้องกัน   
- การกําหนดค่าพอร์ตไฟร์วอลล์   
- การตั้งค่า DNS รวมถึงการค้นหาอัตโนมัติที่จําเป็น กรอบนโยบายผู้ส่ง (SPF) และระเบียน MX (ตามความจําเป็น)     
- การตั้งค่าโฟลว์อีเมลระหว่างสภาพแวดล้อมการส่งข้อความต้นทางและ Exchange Online (ตามความจําเป็น)   
- ดําเนินการย้ายจดหมายจากสภาพแวดล้อมการส่งข้อความต้นทางของคุณไปยัง Office 365   
    > [!NOTE]
    > สําหรับข้อมูลเพิ่มเติมเกี่ยวกับการย้ายข้อมูลและเมล ให้ดูที่[การย้ายข้อมูล](O365-data-migration.md) 
  
![ขั้นตอนการเปิดใช้งานการแลกเปลี่ยนระหว่างขั้นตอนการเปิดใช้งาน](media/O365-Onboarding-Enable-Exchange.png)
  
## <a name="sharepoint-online-and-onedrive-for-business"></a>ออนไลน์ของ SharePoint และ OneDrive สําหรับธุรกิจ

สําหรับ SharePoint แบบออนไลน์และ OneDrive สําหรับธุรกิจ เราให้คําแนะนําสําหรับ: 
- การตั้งค่า DNS   
- การกําหนดค่าพอร์ตไฟร์วอลล์   
- การจัดสรรผู้ใช้และสิทธิ์การใช้งาน   
- การกําหนดค่าคุณลักษณะไฮบริด SharePoint เช่นการค้นหาแบบไฮบริด ไซต์ไฮบริด ระบบภาษี ไฮบริด ชนิดเนื้อหา การสร้างไซต์แบบบริการตนเองแบบไฮบริด (SharePoint Server 2013 เท่านั้น) ตัวเปิดใช้แอปแบบขยาย ไฮบริด OneDrive สําหรับธุรกิจ และไซต์เอกซ์ทราเน็ต
    
ผู้เชี่ยวชาญ FastTrack ให้คําแนะนําเกี่ยวกับการโยกย้ายข้อมูลไปยัง Office 365 โดยใช้การรวมกันของเครื่องมือและเอกสารและโดยการปฏิบัติงานการกําหนดค่าที่สามารถใช้ได้และเป็นไปได้
  
![SharePoint และ Skype สําหรับธุรกิจขั้นตอนการเตรียมพร้อม](media/O365-Onboarding-Enable-SP.png)
  
## <a name="onedrive-for-business"></a>วันไดรฟ์สําหรับธุรกิจ

สําหรับ OneDrive for Business ขั้นตอนจะขึ้นอยู่กับว่าคุณกําลังใช้ SharePoint อยู่ และถ้าเป็นเช่นนั้นรุ่นใด 
  
![ขั้นตอนการเตรียมพร้อมของ OneDrive ระหว่างขั้นตอนการเปิดใช้งาน](media/O365-Onboarding-Enable-ODB.png)
  
## <a name="skype-for-business-online"></a>Skype สําหรับธุรกิจออนไลน์

สําหรับ Skype สําหรับธุรกิจออนไลน์ เราให้คําแนะนําสําหรับ: 
- การกําหนดค่าพอร์ตไฟร์วอลล์    
- การตั้งค่า DNS    
- การสร้างบัญชีสําหรับอุปกรณ์ระบบห้องใด ๆ    
- การปรับใช้ Skype ที่ได้รับการสนับสนุนสําหรับไคลเอ็นต์ออนไลน์ของธุรกิจ

## <a name="microsoft-365-apps"></a>แอปไมโครซอฟท์ 365

สําหรับแอป Microsoft 365 เราให้คําแนะนําสําหรับ: 
- แก้ไขปัญหาการปรับใช้    
- การกําหนดสิทธิ์การใช้งานของผู้ใช้โดยใช้[ศูนย์การจัดการ Microsoft 365](https://go.microsoft.com/fwlink/?linkid=2032704)และ Windows PowerShell    
- การติดตั้งแอป Microsoft 365 จากพอร์ทัล Office 365 โดยใช้คลิกทูรัน    
- การติดตั้งแอป Office สําหรับอุปกรณ์เคลื่อนที่ (เช่น Outlook Mobile, Word Mobile, Excel Mobile และ PowerPoint Mobile) บนอุปกรณ์ iOS หรือ Android ของคุณ   
- การกําหนดค่าการตั้งค่าการปรับปรุงโดยใช้เครื่องมือการปรับใช้ Office 365    
- การตั้งค่าเซิร์ฟเวอร์การแจกจ่ายในไซต์เดียวสําหรับแอป Microsoft 365 รวมถึงความช่วยเหลือในการสร้างแฟ้มconfiguration.xmlสําหรับใช้กับเครื่องมือการปรับใช้ Office 365    
- การปรับใช้โดยใช้ตัวจัดการการตั้งค่าคอนฟิกปลายทางของ Microsoft รวมทั้งความช่วยเหลือเกี่ยวกับการสร้างบรรจุภัณฑ์ตัวจัดการการตั้งค่าคอนฟิกปลายทางของ Microsoft
