# <a name="onboarding-and-migration-phases-for-office-365-us-government"></a>ปฐมนิเทศและการโยกย้ายขั้นตอนสำหรับสำนักงานรัฐบาล๓๖๕สหรัฐอเมริกา

สำนักงาน๓๖๕ปฐมนิเทศมีสี่ขั้นตอนหลัก—เริ่มต้นประเมิน Remediate และเปิดใช้งาน คุณสามารถทำตามขั้นตอนเหล่านี้ด้วยขั้นตอนการโยกย้ายข้อมูลที่เป็นตัวเลือกตามที่แสดงในรูปแบบต่อไปนี้
  
![ขั้นตอนของผลประโยชน์ปฐมนิเทศ](media/O365-Onboarding-Phases.png)
  
สำหรับงานโดยละเอียดสำหรับแต่ละขั้นตอนโปรดดูความ[รับผิดชอบของ FastTrack สำหรับ office ๓๖๕รัฐบาลสหรัฐอเมริกา](US-Gov-appendix-fasttrack-responsibilities.md)และ[รับผิดชอบของคุณสำหรับ office ๓๖๕รัฐบาลสหรัฐอเมริกา](US-Gov-appendix-your-responsibilities.md)
  
## <a name="initiate-phase"></a>ขั้นตอนการเริ่มต้น

หลังจากที่คุณซื้อหมายเลขที่เหมาะสมและชนิดของใบอนุญาตให้ทำตามคำแนะนำจากอีเมลยืนยันการซื้อเพื่อเชื่อมโยงใบอนุญาตกับผู้เช่าที่มีอยู่หรือใหม่ของคุณ 
  
คุณสามารถขอรับความช่วยเหลือผ่านทาง[Microsoft ๓๖๕ admin ศูนย์](https://go.microsoft.com/fwlink/?linkid=2032704)หรือ[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 
 
หากต้องการรับความช่วยเหลือผ่าน[ศูนย์การดูแล Microsoft ๓๖๕](https://go.microsoft.com/fwlink/?linkid=2032704)ผู้ดูแลของคุณจะลงชื่อเข้าใช้ศูนย์การดูแลจากนั้นคลิก**ต้องการความช่วยเหลือ**วิดเจ็ต 

เมื่อต้องการรับความช่วยเหลือผ่าน[เว็บไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698): 
1.  ลงชื่อเข้าใช้ใน[เว็บไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 
2.  เลือก**FastTrack**
3.  เลือก**บริการ**
4.  ทำการ**ร้องขอความช่วยเหลือให้เสร็จสมบูรณ์ด้วย Microsoft ๓๖๕**ฟอร์ม 

นอกจากนี้คู่ค้ายังสามารถรับความช่วยเหลือผ่าน[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698)ในนามของลูกค้าได้ด้วย เมื่อต้องการทำเช่นนั้น:
1.  ลงชื่อเข้าใช้ใน[เว็บไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 
2.  เลือก**FastTrack**
3.  เลือก**ลูกค้าของฉัน**
4.  ค้นหาลูกค้าของคุณหรือเลือกจากรายชื่อลูกค้าของคุณ
5.  เลือก**บริการ**
6.  ทำการ**ร้องขอความช่วยเหลือให้เสร็จสมบูรณ์ด้วย Microsoft ๓๖๕**ฟอร์ม   

นอกจากนี้คุณยังสามารถขอความช่วยเหลือศูนย์ FastTrack จาก[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698)ในรายการบริการที่พร้อมใช้งานสำหรับผู้เช่าของคุณ 
    
ในระหว่างขั้นตอนนี้เราหารือเกี่ยวกับกระบวนการปฐมนิเทศตรวจสอบข้อมูลของคุณและตั้งค่าการประชุมคิก
  
ซึ่งรวมถึงการทำงานร่วมกับคุณเพื่อให้เข้าใจถึงวิธีการที่คุณตั้งใจจะใช้บริการและเป้าหมายขององค์กรและแผนการใช้บริการ
  
![ขั้นตอนการปฐมนิเทศในระหว่างระยะเริ่มต้น](media/O365-Onboarding-Initiate.png)
  
## <a name="assess-phase"></a>ประเมินระยะ

ผู้จัดการ FastTrack ของคุณดำเนินการวางแผนความสำเร็จในการโต้ตอบกับคุณและทีมงานของคุณ ซึ่งจะแนะนำคุณสู่ความสามารถของการบริการที่มีสิทธิ์ที่คุณซื้อ, รากฐานที่สำคัญที่คุณต้องการสำหรับความสำเร็จ, วิธีการในการขับรถการใช้งานของบริการ, และสถานการณ์ที่คุณสามารถใช้เพื่อรับค่าจากบริการ. เราช่วยคุณในการวางแผนความสำเร็จและให้คำติชมเกี่ยวกับพื้นที่สำคัญ (ตามความจำเป็น)
  
ผู้เชี่ยวชาญ FastTrack ทำงานร่วมกับคุณเพื่อประเมินสภาพแวดล้อมต้นทางของคุณและความต้องการ เรามีเครื่องมือสำหรับคุณที่จะรวบรวมข้อมูลเกี่ยวกับสภาพแวดล้อมของคุณและแนะนำคุณผ่านการประมาณความต้องการแบนด์วิดธ์และการประเมินเบราว์เซอร์อินเทอร์เน็ตของคุณระบบปฏิบัติการของลูกค้าระบบชื่อโดเมน (DNS), เครือข่ายโครงสร้างพื้นฐานและระบบข้อมูลประจำตัวเพื่อ ตรวจสอบว่ามีการเปลี่ยนแปลงใดๆที่จำเป็นสำหรับปฐมนิเทศ 
  
เรามีแผนการที่จะนำสภาพแวดล้อมต้นทางของคุณขึ้นอยู่กับข้อกำหนดขั้นต่ำสำหรับการปฐมนิเทศที่ประสบความสำเร็จกับ Office ๓๖๕และถ้าจำเป็นสำหรับกล่องจดหมายที่ประสบความสำเร็จและ/หรือการโยกย้ายข้อมูล เรามีชุดของกิจกรรมที่แนะนำเพื่อเพิ่มมูลค่าของผู้ใช้และการยอมรับ นอกจากนี้เรายังตั้งค่าจุดตรวจสอบที่เหมาะสมสำหรับขั้นตอน Remediate
  
![ขั้นตอนปฐมนิเทศสำหรับการประเมินระยะ](media/O365.png)
  
## <a name="remediate-phase"></a>Remediate เฟส

คุณทำงานด้านต่างๆตามสภาพแวดล้อมของแหล่งที่มาของคุณเพื่อให้คุณตอบสนองความต้องการสำหรับปฐมนิเทศนำไปใช้และย้ายแต่ละบริการตามความจำเป็น
  
![ปฐมนิเทศขั้นตอนในระหว่างเฟส Remediate](media/O365-Onboarding-Remediate.png)
  
นอกจากนี้เรายังมีชุดของกิจกรรมที่แนะนำเพื่อเพิ่มค่าผู้ใช้ปลายทางและการยอมรับ ก่อนที่จะเริ่มขั้นตอนการเปิดใช้งานเราร่วมกันตรวจสอบผลลัพธ์ของกิจกรรมด้านเพื่อให้แน่ใจว่าคุณพร้อมที่จะดำเนินการต่อไป 
  
ในระหว่างขั้นตอนนี้ผู้จัดการ FastTrack ของคุณทำงานร่วมกับคุณในการวางแผนความสำเร็จนำคุณไปยังทรัพยากรที่เหมาะสมและแนวทางปฏิบัติที่ดีที่สุดเพื่อให้คำแนะนำสำหรับคุณที่จะทำให้การบริการที่มีให้กับองค์กรของคุณและการใช้งานไดรฟ์ทั่วบริการ
  
## <a name="enable-phase"></a>เปิดใช้งานเฟส

เมื่อกิจกรรมทั้งหมดเสร็จสมบูรณ์แล้วจะมีการเปลี่ยนแปลงการตั้งค่าโครงสร้างพื้นฐานหลักสำหรับการใช้บริการการจัดเตรียม Office ๓๖๕และการดำเนินกิจกรรมเพื่อขับเคลื่อนการรับบริการ 
  
## <a name="core"></a>หลัก

หลักปฐมนิเทศเกี่ยวข้องกับการจัดเตรียมบริการและผู้เช่าและการรวมข้อมูลเฉพาะตัว นอกจากนี้ยังรวมถึงขั้นตอนสำหรับการให้รากฐานสำหรับบริการปฐมนิเทศเช่นการแลกเปลี่ยนแบบออนไลน์, SharePoint แบบออนไลน์และ Skype สำหรับธุรกิจออนไลน์ คุณและผู้จัดการ FastTrack ของคุณยังคงมีจุดตรวจสอบการวางแผนความสำเร็จในการประเมินความคืบหน้ากับเป้าหมายของคุณและกำหนดสิ่งที่ช่วยเหลือเพิ่มเติมที่คุณต้องการ
  
![หลักปฐมนิเทศขั้นตอนในระหว่างการเปิดใช้งาน phase_1](media/O365-Onboarding-Enable-Core.png)
  
![หลักปฐมนิเทศขั้นตอนในระหว่างการเปิดใช้งาน phase_2](media/204bbb6c-f7fe-4401-aa69-dd828ed44963.png)
  
> [!NOTE]
> WAP ที่ยืนสำหรับพร็อกซีโปรแกรมประยุกต์บนเว็บ SSL ยืนสำหรับชั้น secure Sockets SDS สำหรับการซิงค์ข้อมูลโรงเรียน สำหรับข้อมูลเพิ่มเติมเกี่ยวกับ SDS โปรดดู[ยินดีต้อนรับสู่การซิงค์ข้อมูลโรงเรียนของ Microsoft](https://go.microsoft.com/fwlink/?linkid=871480) 
  
ปฐมนิเทศสำหรับหนึ่งหรือหลายบริการที่มีสิทธิ์สามารถเริ่มต้นเมื่อปฐมนิเทศหลักเสร็จเรียบร้อยแล้ว
  
## <a name="exchange-online"></a>แลกเปลี่ยนแบบออนไลน์

สำหรับการแลกเปลี่ยนแบบออนไลน์เราแนะนำให้คุณทราบถึงกระบวนการเพื่อให้องค์กรของคุณพร้อมที่จะใช้อีเมล ขั้นตอนที่แน่นอนขึ้นอยู่กับสภาพแวดล้อมต้นทางของคุณและแผนการโยกย้ายอีเมลของคุณอาจรวมถึงการให้คำแนะนำสำหรับ:
  
- การตั้งค่าคุณลักษณะของ Exchange แบบออนไลน์ป้องกัน (EOP) สำหรับโดเมนที่เปิดใช้งานจดหมายทั้งหมดที่ตรวจสอบใน Office ๓๖๕   
    > [!NOTE]
    > ระเบียนของคุณแลกเปลี่ยนจดหมาย (MX) ต้องชี้ไปที่ Office ๓๖๕   
- การตั้งค่าคุณลักษณะการแลกเปลี่ยนแบบออนไลน์ขั้นสูงป้องกันภัยคุกคาม (ATP) ถ้าเป็นส่วนหนึ่งของบริการการสมัครใช้งานของคุณเมื่อระเบียน MX ของคุณชี้ไปที่ Office ๓๖๕ คุณลักษณะนี้ถูกกำหนดค่าเป็นส่วนหนึ่งของการตั้งค่า antimalware ป้องกันแบบออนไลน์ของอัตราแลกเปลี่ยน   
- การกำหนดค่าพอร์ตไฟร์วอลล์   
- การตั้งค่า DNS รวมถึงการค้นหาอัตโนมัติที่จำเป็นผู้ส่งกรอบงานนโยบาย (SPF), และระเบียน MX (ตามความจำเป็น)     
- การตั้งค่าการไหลอีเมระหว่างสภาพแวดล้อมการส่งข้อความต้นฉบับของคุณและการแลกเปลี่ยนแบบออนไลน์ (ตามความจำเป็น)   
- ดำเนินการย้ายจดหมายจากสภาพแวดล้อมการส่งข้อความต้นฉบับของคุณไปยัง Office ๓๖๕   
    > [!NOTE]
    > สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการโยกย้ายจดหมายและข้อมูลให้ดูที่การ[ย้ายข้อมูล](O365-data-migration.md) 
  
![แลกเปลี่ยนขั้นตอนปฐมนิเทศในระหว่างขั้นตอนการเปิดใช้งาน](media/O365-Onboarding-Enable-Exchange.png)
  
## <a name="sharepoint-online-and-onedrive-for-business"></a>SharePoint แบบออนไลน์และ OneDrive สำหรับธุรกิจ

สำหรับ SharePoint แบบออนไลน์และ OneDrive สำหรับธุรกิจเราให้คำแนะนำสำหรับ: 
- การตั้งค่า DNS   
- การกำหนดค่าพอร์ตไฟร์วอลล์   
- การจัดเตรียมผู้ใช้และใบอนุญาต   
- การตั้งค่าคอนฟิกลักษณะการทำงานของ SharePoint ไฮบริดสลีเช่นการค้นหาไฮบริดสลีไซต์ไฮบริดสลีระบบภาษีชนิดเนื้อหาการสร้างไซต์ด้วยตนเองแบบไฮบริดสลี (SharePoint Server ๒๐๑๓เท่านั้น) ตัวเปิดใช้แอพลิเคชัน OneDrive สำหรับธุรกิจและไซต์เอ็กซ์ทราเน็ต
    
ผู้เชี่ยวชาญ FastTrack ให้คำแนะนำเกี่ยวกับการย้ายข้อมูลไปยัง Office ๓๖๕โดยใช้การรวมกันของเครื่องมือและเอกสารและการทำงานการกำหนดค่าที่สามารถใช้ได้และเป็นไปได้
  
![SharePoint และ Skype สำหรับขั้นตอนปฐมนิเทศธุรกิจ](media/O365-Onboarding-Enable-SP.png)
  
## <a name="onedrive-for-business"></a>OneDrive สำหรับธุรกิจ

สำหรับ OneDrive สำหรับธุรกิจขั้นตอนต่างๆจะขึ้นอยู่กับถ้าคุณกำลังใช้ SharePoint และในขณะนั้นรุ่นใดก็ตาม 
  
![ขั้นตอนการปฐมนิเทศ OneDrive ในระหว่างระยะการเปิดใช้งาน](media/O365-Onboarding-Enable-ODB.png)
  
## <a name="skype-for-business-online"></a>Skype สำหรับธุรกิจออนไลน์

สำหรับ Skype สำหรับธุรกิจออนไลน์เราให้คำแนะนำสำหรับ: 
- การกำหนดค่าพอร์ตไฟร์วอลล์    
- การตั้งค่า DNS    
- การสร้างบัญชีสำหรับอุปกรณ์ระบบห้องใดๆ    
- การปรับใช้ Skype ที่ได้รับการสนับสนุนสำหรับไคลเอ็นต์ออนไลน์ของธุรกิจ   
- การสร้างการตั้งค่าคอนฟิกเซิร์ฟเวอร์โดเมนแบ่งระหว่าง Lync ๒๐๑๐, Lync ๒๐๑๓หรือ Skype สำหรับสภาพแวดล้อมของธุรกิจ๒๐๑๕เซิร์ฟเวอร์และ Skype สำหรับผู้เช่าออนไลน์ของธุรกิจ (ถ้ามี)
- การเปิดใช้งานแผนการโทรการออกอากาศการประชุม Skype และระบบโทรศัพท์และแผนการโทร (ในตลาดที่พร้อมใช้งาน-ไม่มีอยู่ใน GCC สูงหรือ DoD แผน)

## <a name="yammer-enterprise"></a>Yammer องค์กร

สำหรับ Yammer เราให้คำแนะนำสำหรับการเปิดใช้งานบริการ Yammer องค์กร

> [!NOTE]

## <a name="office-365-proplus"></a>สำนักงาน๓๖๕ ProPlus

สำหรับ Office ๓๖๕ ProPlus เราให้คำแนะนำสำหรับ: 
- จัดการปัญหาการใช้งาน    
- การกำหนดสิทธิ์การใช้งานของ[ผู้ดูแลระบบโดยใช้ศูนย์ดูแล Microsoft ๓๖๕](https://go.microsoft.com/fwlink/?linkid=2032704)และ Windows PowerShell    
- การติดตั้ง Office ๓๖๕ ProPlus จากพอร์ทัล Office ๓๖๕โดยใช้คลิกเรียกใช้    
- การติดตั้งแอปพลิเคชัน Office Mobile (เช่น Outlook Mobile, Word Mobile, Excel Mobile และ PowerPoint Mobile) บนอุปกรณ์เคลื่อนที่ iOS, Android หรือ Windows ของคุณ   
- การกำหนดค่าการปรับปรุงโดยใช้เครื่องมือการปรับใช้ Office ๓๖๕    
- การตั้งค่าเซิร์ฟเวอร์การแจกจ่ายในไซต์เดียวสำหรับ Office ๓๖๕ ProPlus รวมทั้งความช่วยเหลือเกี่ยวกับการสร้างแฟ้ม .xml การตั้งค่าคอนฟิกสำหรับใช้กับเครื่องมือการปรับใช้ Office ๓๖๕    
- การปรับใช้โดยใช้ตัวจัดการการตั้งค่าคอนฟิกศูนย์ระบบ Microsoft รวมถึงความช่วยเหลือด้วยการสร้างบรรจุภัณฑ์ของตัวจัดการการตั้งค่าคอนฟิกศูนย์ระบบ
