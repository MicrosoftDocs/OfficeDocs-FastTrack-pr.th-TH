# <a name="onboarding-and-migration-phases"></a>ขั้นตอนการออนบอร์ดและการโยกย้าย

การปฐมนิเทศ Office ๓๖๕มีสี่ขั้นตอนหลักคือเริ่มต้นประเมินแก้ไขและเปิดใช้งาน คุณสามารถทำตามขั้นตอนเหล่านี้ด้วยขั้นตอนการย้ายข้อมูลที่เป็นตัวเลือกตามที่แสดงในรูปภาพต่อไปนี้
  
![ขั้นตอนของสวัสดิการการปฐมนิเทศ](media/O365-Onboarding-Phases.png)
  
> [!NOTE]
>สำหรับข้อมูลเกี่ยวกับการปฐมนิเทศและการโยกย้ายสำหรับ Office ๓๖๕รัฐบาลสหรัฐฯดู[การปฐมนิเทศและการโยกย้ายสำหรับ office ๓๖๕รัฐบาลสหรัฐอเมริกา](US-Gov-appendix-onboarding-and-migration.md) 

สำหรับงานที่มีรายละเอียดสำหรับแต่ละเฟสให้ดูความรับ[ผิดชอบของ FastTrack](O365-fasttrack-responsibilities.md)และ[หน้าที่รับผิดชอบของคุณ](O365-your-responsibilities.md)
  
## <a name="initiate-phase"></a>เริ่มต้นเฟส

หลังจากที่คุณซื้อหมายเลขและประเภทของใบอนุญาตที่เหมาะสมแล้วให้ทำตามคำแนะนำจากเมลยืนยันการซื้อเพื่อเชื่อมโยงใบอนุญาตกับผู้เช่าที่มีอยู่หรือผู้ให้บริการใหม่ของคุณ 
  
คุณสามารถขอรับความช่วยเหลือผ่าน[ศูนย์ดูแลของ Microsoft ๓๖๕](https://go.microsoft.com/fwlink/?linkid=2032704)หรือ[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 

หากต้องการรับความช่วยเหลือผ่าน[ศูนย์กลางการดูแลระบบ Microsoft ๓๖๕](https://go.microsoft.com/fwlink/?linkid=2032704)ผู้ดูแลของคุณลงชื่อเข้าใช้ศูนย์กลางการดูแลระบบแล้วคลิก "**ต้องการความช่วยเหลือ**" วิดเจ็ต 

เพื่อขอความช่วยเหลือผ่าน[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698): 
1.  เข้าสู่ระบบ[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 
2.  เลือก**บริการ**
3.  ทำการ**ร้องขอความช่วยเหลือด้วยแบบฟอร์ม Microsoft ๓๖๕** 
  
นอกจากนี้พาร์ทเนอร์ยังสามารถรับความช่วยเหลือผ่าน[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698)ในนามของลูกค้าได้อีกด้วย การทำเช่นนั้น:
1.  เข้าสู่ระบบ[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 
2.  เลือก**ลูกค้าของฉัน**
3.  ค้นหาลูกค้าของคุณหรือเลือกจากรายชื่อลูกค้าของคุณ
4.  เลือก**บริการ**
5.  ทำการ**ร้องขอความช่วยเหลือด้วยแบบฟอร์ม Microsoft ๓๖๕**

นอกจากนี้คุณยังสามารถขอความช่วยเหลือจากศูนย์ FastTrack จาก[ไซต์ fasttrack](https://go.microsoft.com/fwlink/?linkid=780698)ในรายการของบริการที่พร้อมใช้งานสำหรับผู้เช่าของคุณ 
    
ในระหว่างขั้นตอนนี้เราจะหารือเกี่ยวกับกระบวนการปฐมนิเทศตรวจสอบข้อมูลของคุณและตั้งค่าการประชุม kickoff ซึ่งรวมถึงการทำงานร่วมกับคุณเพื่อให้เข้าใจว่าคุณต้องการใช้บริการและเป้าหมายขององค์กรและแผนเพื่อกระตุ้นการใช้บริการของคุณอย่างไร
  
![ขั้นตอนการเตรียมการในระหว่างขั้นตอนการเริ่มต้น](media/O365-Onboarding-Initiate.png)
  
## <a name="assess-phase"></a>ประเมินระยะ

ผู้จัดการ FastTrack ของคุณดำเนินการการวางแผนความสำเร็จแบบโต้ตอบกับคุณและทีมงานของคุณ ซึ่งจะแนะนำให้คุณทราบถึงความสามารถของบริการที่มีสิทธิ์ที่คุณซื้อฐานรากที่สำคัญที่คุณต้องการสำหรับความสำเร็จวิธีการในการขับขี่การใช้บริการและสถานการณ์ที่คุณสามารถใช้เพื่อรับค่าจากบริการ เราช่วยคุณในการวางแผนความสำเร็จและให้ข้อเสนอแนะเกี่ยวกับพื้นที่สำคัญ (ตามความจำเป็น)
  
ผู้เชี่ยวชาญ FastTrack ทำงานร่วมกับคุณในการประเมินสภาพแวดล้อมต้นทางและความต้องการของคุณ เรามีเครื่องมือสำหรับคุณในการรวบรวมข้อมูลเกี่ยวกับสภาพแวดล้อมของคุณและแนะนำคุณเกี่ยวกับการประมาณข้อกำหนดของแบนด์วิดท์และการประเมินอินเทอร์เน็ตเบราว์เซอร์ระบบปฏิบัติการของไคลเอ็นต์ระบบชื่อโดเมน (DNS) เครือข่ายโครงสร้างพื้นฐานและระบบการระบุตัวตนเพื่อ ตรวจสอบว่ามีการเปลี่ยนแปลงใดๆที่จำเป็นสำหรับการเตรียมการ 
  
ขึ้นอยู่กับการตั้งค่าปัจจุบันของคุณเราจะให้แผนการแก้ไขที่นำสภาพแวดล้อมแหล่งที่มาของคุณไปถึงความต้องการขั้นต่ำสำหรับการเตรียมใช้งาน Office ๓๖๕และถ้าจำเป็นสำหรับกล่องจดหมายและ/หรือการย้ายข้อมูลที่ประสบความสำเร็จ เราให้ชุดของกิจกรรมที่แนะนำเพื่อเพิ่มมูลค่าผู้ใช้และการยอมรับ นอกจากนี้เรายังตั้งค่าการเรียกจุดตรวจสอบที่เหมาะสมสำหรับระยะแก้ไข
  
![ขั้นตอนการเตรียมการสำหรับขั้นตอนการประเมิน](media/O365.png)
  
## <a name="remediate-phase"></a>ระยะแก้ไข

คุณทำงานด้านการแก้ไขตามสภาพแวดล้อมของแหล่งที่มาของคุณเพื่อให้คุณสามารถตอบสนองความต้องการสำหรับการเตรียมใช้งานและการย้ายข้อมูลแต่ละบริการได้ตามต้องการ
  
![ขั้นตอนการเตรียมการในระหว่างระยะแก้ไข](media/O365-Onboarding-Remediate.png)
  
นอกจากนี้เรายังมีชุดของกิจกรรมที่แนะนำเพื่อเพิ่มมูลค่าผู้ใช้ปลายทางและการยอมรับ ก่อนที่จะเริ่มขั้นตอนการเปิดใช้งานเราร่วมกันตรวจสอบผลลัพธ์ของกิจกรรมการแก้ไขเพื่อให้แน่ใจว่าคุณพร้อมที่จะดำเนินการต่อไป 
  
ในระหว่างขั้นตอนนี้ตัวจัดการ FastTrack ของคุณทำงานร่วมกับคุณในการวางแผนความสำเร็จนำคุณไปยังทรัพยากรที่เหมาะสมและแนวทางปฏิบัติที่ดีที่สุดเพื่อให้คำแนะแนวสำหรับคุณเพื่อให้บริการที่พร้อมใช้งานสำหรับองค์กรของคุณและกระตุ้นการใช้งานทั่วทั้งบริการ
  
## <a name="enable-phase"></a>เปิดใช้งานเฟส

เมื่อกิจกรรมการแก้ไขทั้งหมดเสร็จสมบูรณ์แล้วโฟกัสจะเปลี่ยนไปเพื่อกำหนดค่าโครงสร้างพื้นฐานหลักสำหรับปริมาณการใช้บริการการจัดเตรียม Office ๓๖๕และการดำเนินการกิจกรรมเพื่อกระตุ้นการให้บริการ 
  
## <a name="core"></a>หลัก

การเตรียมการหลักเกี่ยวข้องกับการจัดเตรียมการบริการและผู้เช่าและการรวมตัวตน นอกจากนี้ยังมีขั้นตอนสำหรับการให้รากฐานสำหรับบริการการปฐมนิเทศเช่นการแลกเปลี่ยนแบบออนไลน์, SharePoint แบบออนไลน์และ Skype สำหรับธุรกิจออนไลน์ คุณและผู้จัดการ FastTrack ของคุณยังคงมีความสำเร็จในการวางแผนการประชุมจุดตรวจสอบเพื่อประเมินความคืบหน้ากับเป้าหมายของคุณและกำหนดสิ่งที่ช่วยเหลือเพิ่มเติมที่คุณต้องการ
  
![ขั้นตอนการเตรียมหลักในระหว่างการเปิดใช้งาน phase_1](media/O365-Onboarding-Enable-Core.png)
  
![ขั้นตอนการเตรียมหลักในระหว่างการเปิดใช้งาน phase_2](media/204bbb6c-f7fe-4401-aa69-dd828ed44963.png)
  
> [!NOTE]
> หมายถึงพร็อกซีโปรแกรมประยุกต์บนเว็บ SSL สำหรับการรักษาความปลอดภัยเลเยอร์ซ็อกเก็ต มี SDS ยืนสำหรับการซิงค์ข้อมูลโรงเรียน สำหรับข้อมูลเพิ่มเติมเกี่ยวกับ SDS โปรดดูที่[ยินดีต้อนรับสู่การซิงค์ข้อมูลของโรงเรียน Microsoft](https://go.microsoft.com/fwlink/?linkid=871480) 
  
การเตรียมพร้อมสำหรับบริการที่มีสิทธิ์อย่างน้อยหนึ่งรายการอาจเริ่มต้นเมื่อการเตรียมการหลักเสร็จสิ้น
  
## <a name="exchange-online"></a>แลกเปลี่ยนแบบออนไลน์

สำหรับการแลกเปลี่ยนแบบออนไลน์เราขอแนะนำให้คุณผ่านกระบวนการเพื่อให้องค์กรของคุณพร้อมใช้งานทาง e-mail ขั้นตอนที่แน่นอนขึ้นอยู่กับสภาพแวดล้อมของแหล่งที่มาและแผนการย้ายข้อมูลทาง e-mail ของคุณอาจรวมถึงการให้คำแนะนำสำหรับ:
- การตั้งค่าคุณลักษณะการแลกเปลี่ยนแบบออนไลน์ป้องกัน (EOP) สำหรับโดเมนที่เปิดใช้งานจดหมายทั้งหมดที่ตรวจสอบใน Office ๓๖๕
    > [!NOTE]
    > ระเบียนการแลกเปลี่ยนจดหมาย (MX) ของคุณต้องชี้ไปที่ Office ๓๖๕ 
- การตั้งค่าคุณลักษณะการป้องกันภัยคุกคามขั้นสูงของ Office ๓๖๕ (ATP) ถ้าเป็นส่วนหนึ่งของบริการการสมัครใช้งานของคุณ สำหรับข้อมูลเพิ่มเติมโปรดดูที่[Office ๓๖๕การป้องกันภัยคุกคามขั้นสูง](#office-365-advanced-threat-protection)
- การตั้งค่าคุณลักษณะการป้องกันการสูญหายของข้อมูล (DLP) สำหรับโดเมนที่เปิดใช้งานจดหมายทั้งหมดที่ตรวจสอบใน Office ๓๖๕เป็นส่วนหนึ่งของบริการสมัครสมาชิกของคุณ นี่จะทำเมื่อระเบียน MX ของคุณชี้ไปที่ Office ๓๖๕
- การตั้งค่าการเข้ารหัสลับข้อความ Office ๓๖๕ (OME) สำหรับโดเมนที่เปิดใช้งานจดหมายทั้งหมดที่ตรวจสอบใน Office ๓๖๕เป็นส่วนหนึ่งของบริการสมัครสมาชิกของคุณ นี่จะทำเมื่อระเบียน MX ของคุณชี้ไปที่ Office ๓๖๕

> [!NOTE]
> บริการการจำลองแบบกล่องจดหมาย (MRS) พยายามโยกย้ายจดหมายที่จัดการสิทธิ์ในข้อมูล (IRM) จากกล่องจดหมายในสถานที่ของคุณไปยังกล่องจดหมาย Exchange แบบออนไลน์ที่สอดคล้องกัน ความสามารถในการอ่านเนื้อหาที่มีการป้องกันการโยกย้ายหลังการย้ายขึ้นอยู่กับการแม็ปของลูกค้าและการคัดลอกแม่แบบที่ใช้งานอยู่ไดเรกทอรีสิทธิ์บริการ (AD RMS) ไปยังบริการการจัดการสิทธิ์ Azure (Azure RMS)

- การกำหนดค่าพอร์ตไฟร์วอลล์
- การตั้งค่า DNS รวมถึงการค้นหาอัตโนมัติที่จำเป็นกรอบงานนโยบายผู้ส่ง (SPF) และระเบียน MX (ตามความจำเป็น) 
- การตั้งค่าการไหลของทาง e-mail ระหว่างสภาพแวดล้อมการส่งข้อความต้นฉบับของคุณและการแลกเปลี่ยนแบบออนไลน์ (ตามความจำเป็น)
- ทำการโยกย้ายจดหมายจากสภาพแวดล้อมการส่งข้อความต้นฉบับของคุณไปยัง Office ๓๖๕
- การกำหนดค่าไคลเอ็นต์ของกล่องจดหมาย (Outlook สำหรับ Windows, Outlook บนเว็บและ Outlook สำหรับ iOS และ Android)
    > [!NOTE]
    > สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการ[ย้ายข้อมูลจดหมาย](O365-data-migration.md)และข้อมูล 
  
![ขั้นตอนการเตรียมการแลกเปลี่ยนในระหว่างการเปิดใช้งานระยะ](media/O365-Onboarding-Enable-Exchange.png)
  
## <a name="sharepoint-online-and-onedrive-for-business"></a>SharePoint แบบออนไลน์และ OneDrive สำหรับธุรกิจ

สำหรับ SharePoint แบบออนไลน์และ OneDrive สำหรับธุรกิจเราให้คำแนะนำสำหรับ:
- การตั้งค่า DNS
- การกำหนดค่าพอร์ตไฟร์วอลล์
- การจัดเตรียมผู้ใช้และใบอนุญาต

ผู้เชี่ยวชาญ FastTrack ให้คำแนะนำเกี่ยวกับการย้ายข้อมูลไปยัง Office ๓๖๕โดยใช้การรวมกันของเครื่องมือและเอกสารประกอบและโดยการดำเนินงานการตั้งค่าคอนฟิกที่เกี่ยวข้องและเป็นไปได้

## <a name="office-365-proplus"></a>สำนักงาน๓๖๕ ProPlus

- การจัดการปัญหาการปรับใช้   
- การมอบหมายใบอนุญาตของผู้ใช้ปลายทางโดยใช้[ศูนย์ดูแลของ Microsoft ๓๖๕](https://go.microsoft.com/fwlink/?linkid=2032704)และ Windows PowerShell 
- การติดตั้ง Office ๓๖๕ ProPlus จากเว็บไซต์ Office ๓๖๕โดยใช้คลิก-ทู-รัน   
- การติดตั้งแอป Office บนอุปกรณ์เคลื่อนที่ (เช่น Outlook Mobile, Word Mobile, Excel Mobile และ PowerPoint Mobile) บนอุปกรณ์ iOS, Android หรือ Windows Mobile ของคุณ

- การปรับใช้โดยใช้ตัวจัดการการตั้งค่าคอนฟิกศูนย์ระบบ Microsoft รวมถึงความช่วยเหลือในการสร้างบรรจุภัณฑ์ตัวจัดการการตั้งค่าคอนฟิกศูนย์ระบบ
