# <a name="onboarding-and-migration-phases"></a>ขั้นตอนการออนบอร์ดและการโยกย้าย

การปฐมนิเทศ office 365 มีสี่หลักระยะ — เริ่ม ต้น Assess, Remediate และเปิดใช้งาน คุณสามารถติดตามระยะเหล่านี้ ด้วยระยะการโยกย้ายข้อมูลหรือไม่ก็ได้ดังที่แสดงในภาพต่อไปนี้
  
![ระยะของสวัสดิการปฐมนิเทศ](media/O365-Onboarding-Phases.png)
  
> [!NOTE]
>สำหรับข้อมูลเกี่ยวกับการปฐมนิเทศและโยกย้ายสำหรับ Office 365 เรารัฐบาล ดู[ปฐมนิเทศและการโยกย้ายของรัฐบาลสหรัฐอเมริกาของ Office 365](US-Gov-appendix-onboarding-and-migration.md) 

สำหรับงานรายละเอียดสำหรับแต่ละระยะ ดู[FastTrack ที่รับผิดชอบ](O365-fasttrack-responsibilities.md)และ[ความรับผิดชอบของคุณ](O365-your-responsibilities.md)
  
## <a name="initiate-phase"></a>ขั้นตอนการเริ่มต้น

หลังจากที่คุณซื้อหมายเลขที่เหมาะสมและชนิดของสิทธิ์การใช้งาน ทำตามคำแนะนำจากผู้ซื้อขึ้นอยู่กับสิทธิ์การใช้งานกับผู้เช่าของคุณที่มีอยู่ หรือสร้างการเชื่อมโยง 
  
คุณสามารถขอความช่วยเหลือผ่านทาง[Microsoft 365 admin ศูนย์](https://go.microsoft.com/fwlink/?linkid=2032704)หรือ[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 

เพื่อขอความช่วยเหลือผ่านทาง[Microsoft 365 admin ศูนย์](https://go.microsoft.com/fwlink/?linkid=2032704)admin ของคุณเข้าสู่ศูนย์การดูแล และจากนั้น คลิก**ต้องการความช่วยเหลือได้อย่างไร**widget 

เมื่อต้องการขอความช่วยเหลือผ่าน[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698): 
1.  เข้าสู่ระบบไปยัง[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 
2.  เลือก**บริการ**
3.  แบบฟอร์ม**ร้องขอความช่วยเหลือเกี่ยวกับ Microsoft 365**ให้เสร็จสมบูรณ์ 
  
คู่ค้าสามารถจะขอความช่วยเหลือผ่าน[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698)ในนามของลูกค้า เมื่อต้องการทำเช่นนี้:
1.  เข้าสู่ระบบไปยัง[ไซต์ FastTrack](https://go.microsoft.com/fwlink/?linkid=780698) 
2.  เลือก**ลูกค้าของฉัน**
3.  ค้นหาลูกค้าของคุณ หรือเลือกจากรายการของลูกค้าของคุณ
4.  เลือก**บริการ**
5.  แบบฟอร์ม**ร้องขอความช่วยเหลือเกี่ยวกับ Microsoft 365**ให้เสร็จสมบูรณ์

คุณสามารถขอให้ขอความช่วยเหลือศูนย์ FastTrack จาก[FastTrack ไซต์](https://go.microsoft.com/fwlink/?linkid=780698)ในรายการของบริการที่พร้อมใช้งานสำหรับผู้เช่าของคุณ 
    
ในระหว่างขั้นตอนนี้ เราอภิปรายกระบวนการปฐมนิเทศ การตรวจสอบข้อมูลของคุณ และตั้งค่าการประชุม kickoff ซึ่งรวมถึงการทำงานกับคุณเพื่อให้เข้าใจถึงวิธีการที่คุณต้องการใช้บริการขององค์กรเป้าหมาย และแผนการใช้บริการของไดรฟ์
  
![ขั้นตอนการปฐมนิเทศระหว่างขั้นตอนการเริ่มต้น](media/O365-Onboarding-Initiate.png)
  
## <a name="assess-phase"></a>ขั้นตอนประเมิน

ผู้จัดการของคุณ FastTrack ดำเนินการสำเร็จแบบโต้ตอบที่ติดต่อกับคุณและทีมของคุณเริ่มการวางแผน ซึ่งนำคุณไปความสามารถของสิทธิ์ได้รับบริการคุณซื้อ มูลนิธิคีย์ที่คุณต้องการสำหรับความสำเร็จ ระเบียบวิธีสำหรับการขับใช้การบริการ และสถานการณ์สมมติที่คุณสามารถใช้เพื่อเรียกค่าจากบริการ เราช่วยคุณในการวางแผนความสำเร็จ และให้ผลตอบสนองในเรื่องหลัก (จำเป็น)
  
ผู้เชี่ยวชาญ FastTrack ทำงานกับคุณสามารถประเมินของสภาพแวดล้อมของแหล่งที่มาและความต้องการ เราให้เครื่องมือเพื่อให้คุณสามารถรวบรวมข้อมูลเกี่ยวกับสภาพแวดล้อมของคุณ และแนะนำคุณเกี่ยวกับการประมาณแบนด์วิดท์ของความต้องการ และการประเมินของอินเทอร์เน็ตเบราว์เซอร์ ระบบปฏิบัติการของไคลเอนต์ Domain Name System (DNS), เครือข่าย โครงสร้างพื้นฐาน และระบบรหัสประจำตัว ตรวจสอบว่า การเปลี่ยนแปลงใด ๆ ที่จำเป็นสำหรับการปฐมนิเทศ 
  
ขึ้นอยู่กับการตั้งค่าปัจจุบันของคุณ เราให้แผนด้านที่เปิดแหล่งที่มาของสภาพแวดล้อมจนถึงความต้องการขั้นต่ำสำหรับการปฐมนิเทศเสร็จเรียบร้อยแล้วเพื่อ Office 365 และ ถ้าจำ เป็น สำหรับการโยกย้ายกล่องจดหมายและ/หรือข้อมูลที่ประสบความสำเร็จ เราให้ชุดของกิจกรรมที่แนะนำเมื่อต้องการเพิ่มค่าของผู้ใช้ปลายทางและเริ่ม เรายังได้เรียกจุดตรวจสอบที่เหมาะสมสำหรับขั้นตอนการ Remediate
  
![ขั้นตอนการปฐมนิเทศสำหรับเฟส Assess](media/O365.png)
  
## <a name="remediate-phase"></a>Remediate ขั้นตอน

คุณทำงานด้านขึ้นอยู่กับสภาพแวดล้อมต้นทางของคุณเพื่อให้คุณตอบสนองความต้องการสำหรับการปฐมนิเทศ ใช้ และโยกย้ายแต่ละการบริการตามความจำเป็น
  
![ขั้นตอนการปฐมนิเทศระหว่างขั้นตอนการ Remediate](media/O365-Onboarding-Remediate.png)
  
เรายังมีชุดของกิจกรรมที่แนะนำเมื่อต้องการเพิ่มค่าของผู้ใช้ปลายทางและเริ่ม ก่อนที่จะเริ่มขั้นตอนการเปิดใช้งาน เรานิติบุคคลตรวจสอบผลลัพธ์ของกิจกรรมด้านเพื่อให้แน่ใจว่า คุณพร้อมที่จะดำเนินต่อหรือไม่ 
  
ในระหว่างขั้นตอนนี้ ผู้จัดการ FastTrack ของคุณทำงานร่วมกับคุณบนความสำเร็จในการวางแผน การแนะนำให้คุณไปยังทรัพยากรที่ถูกต้องและวิธีปฏิบัติที่ดีเพื่อให้คำแนะนำสำหรับการให้บริการสามารถใช้องค์กรของคุณและการใช้งานไดรฟ์ข้ามบริการคุณ
  
## <a name="enable-phase"></a>ขั้นตอนการเปิดใช้งาน

เมื่อกิจกรรมด้านทั้งหมดเสร็จสมบูรณ์ โฟกัสเลื่อนเพื่อตั้งค่าคอนฟิกโครงสร้างพื้นฐานหลักสำหรับปริมาณการใช้การบริการ การเตรียมใช้งาน Office 365 และทำกิจกรรมที่จะเริ่มบริการของไดรฟ์ 
  
## <a name="core"></a>หลัก

การปฐมนิเทศหลักเกี่ยวข้องกับการรวมการเตรียมใช้งาน และผู้เช่า และข้อมูลเฉพาะตัวของบริการ นอกจากนี้ยังรวมถึงขั้นตอนที่ให้พื้นฐานสำหรับการบริการปฐมนิเทศเช่น Exchange ออนไลน์ ออนไลน์ของ SharePoint และ Skype สำหรับธุรกิจออนไลน์ ผู้จัดการของคุณ FastTrack และคุณยังได้วางแผนการประชุมจุดตรวจสอบการประเมินความคืบหน้าเทียบกับเป้าหมายของคุณ และความช่วยเหลือเพิ่มเติมใดที่คุณต้องการตรวจสอบความสำเร็จ
  
![ขั้นตอนการปฐมนิเทศหลักระหว่าง phase_1 เปิดใช้งาน](media/O365-Onboarding-Enable-Core.png)
  
![ขั้นตอนการปฐมนิเทศหลักระหว่าง phase_2 เปิดใช้งาน](media/204bbb6c-f7fe-4401-aa69-dd828ed44963.png)
  
> [!NOTE]
> WAP หมายถึงสำหรับพร็อกซีโปรแกรมประยุกต์บนเว็บ SSL หมายถึง การรักษาความปลอดภัย Secure Sockets Layer SDS หมายถึงสำหรับซิงค์ข้อมูลโรงเรียน สำหรับข้อมูลเพิ่มเติมเกี่ยวกับ SDS ดู[ยินดีต้อนรับสู่ Microsoft ซิงค์ข้อมูลของโรงเรียน](https://go.microsoft.com/fwlink/?linkid=871480) 
  
ปฐมนิเทศสำหรับอย่าง น้อยหนึ่งบริการที่เหมาะสมสามารถเริ่มต้นหลังจากเสร็จสิ้นการปฐมนิเทศหลักเท่านั้น
  
## <a name="exchange-online"></a>การแลกเปลี่ยนแบบออนไลน์

สำหรับการแลกเปลี่ยนแบบออนไลน์ เราแนะนำคุณผ่านขั้นตอนการเรียกองค์กรของคุณพร้อมสำหรับการใช้อีเมล ขั้นตอนที่แน่นอน ขึ้นอยู่กับสภาพแวดล้อมต้นทางของคุณและวางแผนการย้ายอีเมล อาจรวมถึงการให้คำแนะนำสำหรับ:
- การตั้งค่าคุณลักษณะการป้องกันแบบออนไลน์ของอัตราแลกเปลี่ยน (EOP) สำหรับการเปิดใช้งานจดหมายโดเมนทั้งหมดใน Office 365
    > [!NOTE]
    > เร็กคอร์ดของการแลกเปลี่ยน (MX) จดหมายคุณต้องชี้ไป Office 365 
- การตั้งค่าคุณลักษณะการแลกเปลี่ยนแบบออนไลน์ขั้นสูงคุกคามป้องกัน (ATP) ถ้าเป็นส่วนหนึ่งของบริการสมัครใช้งานของคุณ ซึ่งจะทำได้เมื่อระเบียน MX ของคุณชี้ไปที่ Office 365 คุณลักษณะนี้ถูกกำหนดค่าเป็นส่วนหนึ่งของการตั้งค่า antimalware ป้องกันแบบออนไลน์ของอัตราแลกเปลี่ยน
- การตั้งค่าคุณลักษณะการป้องกัน (DLP) สูญเสียข้อมูลสำหรับการเปิดใช้งานจดหมายโดเมนทั้งหมดใน Office 365 การตรวจสอบเป็นส่วนหนึ่งของบริการสมัครใช้งานของคุณ ซึ่งจะทำได้เมื่อระเบียน MX ของคุณชี้ไปที่ Office 365
- การตั้งค่า Office 365 ข้อความเข้ารหัสลับ (าน) สำหรับการเปิดใช้งานจดหมายโดเมนทั้งหมดใน Office 365 การตรวจสอบเป็นส่วนหนึ่งของบริการสมัครใช้งานของคุณ ซึ่งจะทำได้เมื่อระเบียน MX ของคุณชี้ไปที่ Office 365
- ตั้งค่าคอนฟิกพอร์ตไฟร์วอลล์
- การตั้งค่า DNS รวมถึงการค้นหาอัตโนมัติที่จำเป็น โครงสร้างนโยบายของผู้ส่ง (SPF), และระเบียน MX (จำเป็น) 
- การตั้งค่าอีเมลกระแสระหว่างแหล่งที่มาของข้อความแวดล้อมและการแลกเปลี่ยนแบบออนไลน์ (จำเป็น)
- Undertaking การย้ายจดหมายจากแหล่งที่มาของสภาพแวดล้อมของ Office 365 การส่งข้อความ
- การกำหนดค่าไคลเอ็นต์จดหมาย (Outlook สำหรับ Windows, Outlook บนเว็บ และ Outlook สำหรับ iOS และ Android)
    > [!NOTE]
    > สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการย้ายจดหมายและข้อมูล ดู[การโยกย้ายข้อมูล](O365-data-migration.md) 
  
![ขั้นตอนการปฐมนิเทศอัตราแลกเปลี่ยนระหว่างขั้นตอนการเปิดใช้งาน](media/O365-Onboarding-Enable-Exchange.png)
  
## <a name="sharepoint-online-and-onedrive-for-business"></a>SharePoint แบบออนไลน์และ OneDrive สำหรับธุรกิจ

สำหรับ SharePoint แบบออนไลน์และ OneDrive สำหรับธุรกิจ เราให้คำแนะนำสำหรับ:
- การตั้งค่า DNS
- ตั้งค่าคอนฟิกพอร์ตไฟร์วอลล์
- การเตรียมใช้งานผู้ใช้และสิทธิ์การใช้งาน   
- การตั้งค่าคอนฟิกไฮบริดสลีคุณลักษณะของ SharePoint เช่นเดียวกับค้นหาไฮบริดสลี ไซต์ไฮบริดสลี ระบบภาษีไฮบริดสลี ชนิดเนื้อหา การสร้างไซต์ด้วยตนเองไฮบริดสลี (SharePoint Server 2013 เท่านั้น), ขยายตัวเปิดใช้โปรแกรมประยุกต์ ไฮบริดสลี OneDrive สำหรับธุรกิจ และไซต์เอ็กซ์ทราเน็ต
    
ผู้เชี่ยวชาญ FastTrack ให้คำแนะนำในการโยกย้ายข้อมูลกับ Office 365 โดยใช้การรวมกันของเครื่องมือและเอกสารประกอบ และ ด้วยการดำเนินงานการกำหนดค่าเป็นไปได้ และสามารถใช้

## <a name="project-online-professional-and-project-online-premium"></a>โครงการ Professional ออนไลน์และค่าจ้างพิเศษออนไลน์ของโครงการ

สำหรับผู้เชี่ยวชาญออนไลน์ของโครงการและโครงการพิเศษที่ออนไลน์ เราให้คำแนะนำสำหรับ:

- การเชื่อมต่อโครงการออนไลน์เดสก์ท็อปไคลเอ็นต์ไปยังโครงการ Professional ออนไลน์หรือค่าจ้างพิเศษออนไลน์ของโครงการ

สำหรับ Office 365 ProPlus เราให้คำแนะนำสำหรับ:
