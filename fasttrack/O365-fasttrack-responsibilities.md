# <a name="fasttrack-responsibilities"></a>ความรับผิดชอบของ FastTrack

ผู้เชี่ยวชาญ FastTrack มีความรับผิดชอบต่อไปนี้ในระหว่างปฐมนิเทศ\*
  
## <a name="general"></a>ทั่ว ไป

- ให้ความช่วยเหลือในการสนับสนุนระยะไกลกับคุณในการพัฒนาวางแผนความสำเร็จและการดำเนินการและสำหรับกิจกรรมการกำหนดค่าที่จำเป็นตามรายละเอียดในคำอธิบายขั้นตอน
- มีเอกสารประกอบและเครื่องมือซอฟต์แวร์ที่พร้อมใช้งานคอนโซลผู้ดูแลระบบและสคริปต์เพื่อให้คำแนะนำสำหรับคุณในการลดหรือกำจัดงานการตั้งค่าคอนฟิกและทรัพยากรการวางแผนความสำเร็จ 
    
## <a name="initiate-phase"></a>ขั้นตอนการเริ่มต้น

- ทำงานร่วมกับคุณเพื่อให้เข้าใจถึงความตั้งใจเป้าหมายองค์กรและแผนการใช้บริการของคุณ
- ทำงานกับคุณโดยใช้ Office ๓๖๕บริการร่วมมือกัน (เช่นทีม Microsoft) เพื่อเริ่มต้นปฐมนิเทศ   
- กำหนดบริการที่มีสิทธิ์ที่คุณต้องการไปยังบนเครื่อง 
    
## <a name="assess-phase"></a>ประเมินระยะ

- ถือการวางแผนความสำเร็จเพื่อให้คำแนะนำกับคุณสำหรับการยอมรับผู้ใช้ที่ประสบความสำเร็จ  
- แสดงภาพรวมของผู้ดูแล
- ให้คำแนะนำเกี่ยวกับ: 
  - ระบบชื่อโดเมน (DNS), เครือข่ายและความต้องการโครงสร้างพื้นฐาน
  - ความต้องการของลูกค้า (อินเทอร์เน็ตเบราว์เซอร์, ระบบปฏิบัติการไคลเอนต์, อุปกรณ์มือถือและความต้องการของบริการ)
  - ข้อมูลเฉพาะตัวของผู้ใช้และการจัดเตรียม
  - การเปิดใช้งานบริการที่มีสิทธิ์ที่จะซื้อและกำหนดเป็นส่วนหนึ่งของปฐมนิเทศ 
  - การขับขี่ที่ประสบความสำเร็จในการรับบริการและมูลค่า  
- สร้างเส้นเวลาสำหรับกิจกรรมด้าน
- แสดงรายการตรวจสอบด้าน 
- ประเมินการ SharePoint Server ๒๐๑๓หรือโครงสร้างพื้นฐานของ SharePoint Server ๒๐๑๖ที่มีอยู่รวมถึง:
  - ข้อกำหนดเบื้องต้นสำหรับไฮบริดสลี SharePoint แบบออนไลน์ 
  - ความพร้อมของโครงสร้างพื้นฐานในสถานที่สำหรับคุณลักษณะไฮบริดสลี SharePoint แบบออนไลน์
  - การเข้าถึงปลายทางแบบออนไลน์ของ SharePoint ที่จำเป็น
  - ผู้ชมสำหรับ OneDrive สำหรับธุรกิจไฮบริดสลี 
- ประเมิน Lync ที่มีอยู่ Skype สำหรับธุรกิจออนไลน์หรือโครงสร้างพื้นฐานของทีม Microsoft รวมถึง:
  - Skype ที่สนับสนุนสำหรับไคลเอ็นต์ธุรกิจหรือทีมกลยุทธ์การปรับใช้ไคลเอนต์
  - การเข้าถึงปลายทาง
  - คุณภาพการเชื่อมต่อ
  - การประมาณแบนด์วิดธ์
  - ข้อกำหนดเบื้องต้นเพื่อสนับสนุนการกำหนดค่าเซิร์ฟเวอร์โดเมนแยก
  - ความพร้อมของผู้ใช้ที่ระบุที่จะย้ายไปยัง Skype สำหรับธุรกิจออนไลน์หรือทีม
- ประเมินโครงสร้างพื้นฐานการส่งข้อความรวมถึง: 
  - การไหลของจดหมายโดยรวมและหลักการกำหนดเส้นทาง
  - การเข้าถึงไคลเอ็นต์ (รวมถึงปลายทางการเข้าถึงไคลเอ็นต์ที่เผยแพร่แล้ว)
  - สภาพแวดล้อมการส่งข้อความต้นฉบับสำหรับการรวมที่ต้องการ
- ให้ย้ายข้อมูลถ้ามีการใช้บริการการย้ายข้อมูลศูนย์ FastTrack และถ้าคุณพบสิทธิ์
    
## <a name="remediate-phase"></a>Remediate เฟส

- ถือการประชุมสายกับคุณตามกำหนดการที่ตกลงกันเพื่อตรวจสอบความคืบหน้าของกิจกรรมด้านและการวางแผนความสำเร็จ 
- แนะนำคุณผ่านการทำงานเครื่องมือประเมินเพื่อระบุและ remediate ปัญหาและตีความผล
    
## <a name="enable-phase"></a>เปิดใช้งานเฟส

ให้คำแนะนำเกี่ยวกับ: 
- ประเมินความคืบหน้าในการวางแผนความสำเร็จและการกำหนดสิ่งที่ช่วยเหลือเพิ่มเติมที่คุณต้องการ
- เปิดใช้งาน Office ๓๖๕ผู้เช่าของคุณ  
- การกำหนดค่าโพรโทคอล TCP/IP และพอร์ตไฟร์วอลล์
- การกำหนดค่า DNS สำหรับบริการที่มีสิทธิ์ 
- ตรวจสอบการเชื่อมต่อกับ Office ๓๖๕
- การเชื่อมต่อไดเรกทอรีที่ใช้งานอยู่ในสถานที่กับ Azure ไดเรกทอรีที่ใช้งานอยู่:
  - การติดตั้งเซิร์ฟเวอร์การซิงโครไนส์ไดเรกทอรีระหว่างบริการการโดเมนไดเรกทอรีที่ใช้งานอยู่ (AD DS) และ Office ๓๖๕ถ้าจำเป็น 
  - การกำหนดค่าการซิงโครไนส์รหัสผ่าน (แฮรหัสผ่าน) ไปยัง Office ๓๖๕ (Azure ไดเรกทอรีที่ใช้งานอยู่) ด้วยเครื่องมือ Azure เชื่อมต่อไดเรกทอรีที่ใช้งานอยู่ถ้าจำเป็น
  - สำหรับสภาพแวดล้อมแบบเดี่ยวและแบบหลายป่า:
      - การกำหนดค่าการรับรองความถูกต้องแบบพาส-ทรูไดเรกทอรีที่ใช้งานอยู่ Azure\*\*
      - การกำหนดค่า Azure ไดเรกทอรีที่ใช้งานอยู่อย่างราบรื่นเข้าสู่ระบบแบบครั้งเดียว (SSO), ถ้าจำเป็น\*\*\*
    > [!NOTE]
    > การรับรองความถูกต้องแบบพาส-ทรูไดเรกทอรีที่ใช้งานอยู่ Azure สำหรับสภาพแวดล้อมของฟอเรสต์หลายได้รับการสนับสนุนถ้ามี trusts ฟอเรสต์ระหว่างฟอเรสต์ไดเรกทอรีที่ใช้งานอยู่ของคุณและถ้ามีการกำหนดเส้นทางส่วนต่อท้ายของชื่อ ตัวแทนเพิ่มเติมสามารถติดตั้งบนเซิร์ฟเวอร์ในสถานที่หลายเพื่อให้พร้อมใช้งานสูงสำหรับการร้องขอการเข้าสู่ระบบ สำหรับข้อมูลเพิ่มเติมให้ดูการ[รับรองความถูกต้องแบบพาส-ทรูไดเรกทอรีที่ใช้งานอยู่ azure: เริ่มต้นอย่างรวดเร็ว](https://go.microsoft.com/fwlink/?linkid=860094)และ[Azure ไดเรกทอรีที่ใช้งานอยู่โดยไม่ต้องเข้าสู่ระบบแบบครั้งเดียว:](https://go.microsoft.com/fwlink/?linkid=860095) 
- สำหรับฟอเรสต์เดียวเมื่อผู้ใช้อีเมลที่ติดต่อกับภายนอกเป็นเป้าหมาย: 
  - การติดตั้งและการตั้งค่าคอนฟิก FS โฆษณาสำหรับการรับรองความถูกต้องโดเมนเฉพาะที่มี Office ๓๖๕ในการกำหนดค่าการทนต่อข้อบกพร่องในไซต์เดียว
  - การติดตั้งและการกำหนดค่า WAP สำหรับการเผยแพร่โครงสร้างพื้นฐาน FS โฆษณาของคุณไปยังอินเทอร์เน็ตเมื่อจำเป็น
    > [!NOTE]
    > สำหรับการกำหนดค่าทั้งหมด forests การปรับใช้ FS โฆษณาจะออกจากขอบเขต 
- การทดสอบฟังก์ชัน SSO ที่ไม่มีรอยต่อถ้าจัดวาง
- การขับขี่ที่ประสบความสำเร็จในการรับบริการและมูลค่า
    
\*\*สำหรับข้อมูลเพิ่มเติมเกี่ยวกับขีดจำกัดการรับรองความถูกต้องทรูให้ดูที่การ[รับรองความถูกต้องแบบพาส-ทรู Azure ไดเรกทอรีที่ใช้งานอยู่: ข้อจำกัดปัจจุบัน](https://go.microsoft.com/fwlink/?linkid=860356) 

\*\*\*สำหรับข้อมูลเพิ่มเติมเกี่ยวกับปัญหา SSO ไม่ราบรื่นให้ดู[ที่การแก้ไขปัญหา Azure ไดเรกทอรีที่ใช้งานอยู่โดยไม่ต้องเข้าสู่ระบบแบบครั้งเดียว](https://go.microsoft.com/fwlink/?linkid=841926)

## <a name="exchange-online"></a>แลกเปลี่ยนแบบออนไลน์

ให้คำแนะนำเกี่ยวกับ:
- การสร้างหรือการปรับปรุงระเบียน DNS 
- การเปิดใช้งานการกำหนดเส้นทางอีเมลระหว่างระบบการส่งข้อความต้นฉบับและสภาพแวดล้อมของ Office ๓๖๕ 
- การตั้งค่าคอนฟิกการป้องกันแบบออนไลน์ของอัตราแลกเปลี่ยน, การป้องกันข้อมูลสูญหาย (DLP) และ Office ๓๖๕การเข้ารหัสลับข้อความ (ATP), และ Office ๓๖๕ป้องกันการคุกคามขั้นสูง (OME) (ถ้ามีในการสมัครใช้งานของคุณ๓๖๕) โดเมนที่เปิดใช้งานจดหมาย
- การกำหนดค่าการตั้งค่าไฮบริดสลีระหว่างองค์กรแลกเปลี่ยนในสถานที่เดียวและ Office ๓๖๕*หรือ*ระหว่างหลายองค์กรแลกเปลี่ยนในสถานที่และ office ๓๖๕ 
- การกำหนดค่าไคลเอ็นต์ของกล่องจดหมาย (Outlook สำหรับ Windows, Outlook บนเว็บและ Outlook สำหรับ iOS และ Android)
- การกำหนดค่าอัตโนมัติการสืบสวนและการตอบสนองสำหรับ Office ๓๖๕ ATP (ถ้ามีในการสมัครใช้งานของคุณ)
    
สำหรับข้อมูลเพิ่มเติมเกี่ยวกับความรับผิดชอบในการย้ายข้อมูลให้ดูที่การ[ย้ายข้อมูล](O365-data-migration.md)
  
## <a name="microsoft-teams"></a>Microsoft Teams

ให้คำแนะนำเกี่ยวกับ:
- ยืนยันความต้องการขั้นต่ำ
- การกำหนดค่าพอร์ตไฟร์วอลล์
- การตั้งค่า DNS  
- การยืนยันกลุ่มคนถูกเปิดใช้งานบน Office ๓๖๕ผู้เช่าของคุณ
- การเปิดหรือปิดใช้งานใบอนุญาต
- การกระจายของลูกค้ากลุ่ม
- คุณสมบัตินี้ pro และผู้ดูแลระบบ
- คุณสมบัติของผลิตภัณฑ์หลัก
- แม่แบบความสำเร็จของลูกค้า
- การสร้างบัญชีที่จะเชื่อมโยงกับอุปกรณ์ระบบห้องที่ได้รับการสนับสนุน (สูงสุด10บัญชี) 
- การเปิดใช้งานการกำหนดเส้นทางโดยตรง
- การเปิดใช้งานการประชุมทางเสียง
- การตั้งค่าองค์กรสำหรับการประชุมที่เป็นค่าเริ่มต้นของบริดจ์
- การกำหนดบริดจ์การประชุมให้กับผู้ใช้ที่ได้รับอนุญาต
- การเปิดใช้งานระบบโทรศัพท์
- การเปิดใช้งานระบบโทรศัพท์และแผนการโทรปฐมนิเทศ (ในตลาดที่มีอยู่)
- การกำหนดหมายเลขให้กับผู้ใช้ที่ได้รับอนุญาต
- หมายเลขท้องถิ่น porting คำแนะนำผ่าน UI ถึง๙๙๙
- การสนับสนุนหมายเลขท้องถิ่น porting SR มากกว่า๙๙๙ 
- การเปิดใช้งานการแข่งขันที่มีอยู่ในทีม 
- การตั้งค่าองค์กรและการรวมเข้ากับกระแสข้อมูลของ Microsoft

## <a name="office-365-advanced-threat-protection"></a>การป้องกันการคุกคามขั้นสูงของ Office ๓๖๕

ให้คำแนะนำเกี่ยวกับ:
- การเปิดใช้งานการเชื่อมโยงที่ปลอดภัย
- การเปิดใช้งานเอกสารแนบที่ปลอดภัย
- การเปิดใช้งานนโยบายการป้องกันฟิชชิ่ง
- การกำหนดค่าระบบอัตโนมัติการสืบสวนและการตอบสนอง
- ใช้การจำลองการโจมตี
- การรายงานและการวิเคราะห์ภัยคุกคาม

## <a name="office-365-proplus"></a>สำนักงาน๓๖๕ ProPlus

ให้คำแนะนำเกี่ยวกับ:
- จัดการปัญหาการใช้งาน
- การกำหนดสิทธิ์การใช้งานของ[ผู้ดูแลระบบโดยใช้ศูนย์ดูแล Microsoft ๓๖๕](https://go.microsoft.com/fwlink/?linkid=2032704)และ Windows PowerShell
- การติดตั้ง Office ๓๖๕ ProPlus จากพอร์ทัล Office ๓๖๕โดยใช้คลิกเรียกใช้
- การติดตั้งแอปพลิเคชัน Office Mobile (เช่น Outlook สำหรับ iOS และ Android, Word Mobile, Excel Mobile และ PowerPoint Mobile) บนอุปกรณ์เคลื่อนที่ iOS, Android หรือ Windows ของคุณ 
- การกำหนดค่าการปรับปรุงโดยใช้เครื่องมือการปรับใช้ Office ๓๖๕
- การเลือกและตั้งค่าการติดตั้งท้องถิ่นหรือเมฆ
- การสร้างการกำหนดค่าเครื่องมือการปรับใช้ Office XML ด้วยเครื่องมือกำหนดเองของ Office หรือ XML ดั้งเดิมเพื่อกำหนดค่าแพคเกจการปรับใช้
- การปรับใช้โดยใช้ตัวจัดการการตั้งค่าคอนฟิกศูนย์ระบบรวมถึงความช่วยเหลือด้วยการสร้างบรรจุภัณฑ์ของตัวจัดการการตั้งค่าคอนฟิกศูนย์ระบบ
    
## <a name="onedrive-for-business"></a>OneDrive สำหรับธุรกิจ

ให้คำแนะนำเกี่ยวกับ:
- การระบุรุ่นของ SharePoint และตัวเลือกการรวมในสถานที่ 
- การระบุตัวเลือกการซิงโครไนส์และข้อมูลประจำตัว
- การเลือกตัวเลือกไวร์:   
  - ไวร์เพียงครั้งเดียว
  - ไวร์ตามขั้นตอน (เรียงลำดับและเลิก)
- กำลังเตรียมสภาพแวดล้อมในสถานที่สำหรับ OneDrive สำหรับการปรับใช้ธุรกิจ:
  - การระบุ OneDrive ที่ถูกต้องสำหรับไคลเอ็นต์การซิงค์ธุรกิจ
  - การกำหนดค่า DNS พอร์ตเครือข่ายและไฟร์วอลล์ 
- การกำหนดใบอนุญาตของผู้ใช้ปลาย 
- การตั้งค่ากลุ่มเป้าหมาย SharePoint Online เพื่อควบคุมและควบคุมผู้ที่ได้รับ OneDrive สำหรับธุรกิจ 
- การปรับใช้ OneDrive สำหรับไคลเอ็นต์ซิงค์ธุรกิจกับเดสก์ท็อป   
- วิธีการกำหนดค่า SharePoint แบบออนไลน์ไฮบริดสลี OneDrive สำหรับการเปลี่ยนเส้นทางธุรกิจ (SharePoint ๒๐๑๓และ SharePoint ๒๐๑๖เท่านั้น)
- การย้ายข้อมูลถ้ามีใช้บริการการย้ายข้อมูลศูนย์ FastTrack และถ้าคุณพบสิทธิ์
    
## <a name="outlook-for-ios-and-android"></a>Outlook สำหรับ iOS และ Android

ให้คำแนะนำเกี่ยวกับ:
- ดาวน์โหลด Outlook บนอุปกรณ์ iOS และ Android
- การกำหนดค่าบัญชีอีเมลภายใน Outlook

## <a name="power-bi"></a>พลังงาน BI

ให้คำแนะนำเกี่ยวกับ:
- ตรวจทานแผนการสมัครใช้งาน BI Power 
- การเพิ่มบริการการใช้พลังงาน BI 
- การดาวน์โหลดแอปเดสก์ท็อป Power BI
    
## <a name="project-online"></a>โครงการออนไลน์

ให้คำแนะนำเกี่ยวกับ:
- ตรวจทานแผนการสมัครใช้งาน
- กำลังตรวจสอบฟังก์ชันการทำงานพื้นฐานของ SharePoint
- การเพิ่มบริการออนไลน์ของโครงการ
- การเพิ่มผู้ใช้ลงในโครงการแบบออนไลน์รวมถึงการซิงค์ ERP
- กำลังตรวจสอบฟังก์ชันการทำงานแบบออนไลน์ของโครงการพื้นฐานโดยการสร้างโครงการ
    
## <a name="project-online-professional-and-project-online-premium"></a>พรีเมี่ยมออนไลน์ของโครงการระดับมืออาชีพและโครงการ

ให้คำแนะนำเกี่ยวกับ:
- จัดการปัญหาการใช้งาน
- การกำหนดสิทธิ์การใช้งานของ[ผู้ดูแลระบบโดยใช้ศูนย์ดูแล Microsoft ๓๖๕](https://go.microsoft.com/fwlink/?linkid=2032704)และ Windows PowerShell
- ดาวน์โหลดและติดตั้งไคลเอ็นต์เดสก์ท็อปของโครงการออนไลน์จากพอร์ทัล   
- การตั้งค่าคอนฟิกการปรับปรุงโดยใช้เครื่องมือการปรับใช้ Office ๓๖๕หรือแม่แบบของนโยบายกลุ่ม
- การตั้งค่าเซิร์ฟเวอร์การแจกจ่ายบนไซต์เดียวสำหรับไคลเอ็นต์เดสก์ทอป Project Online รวมทั้งคำแนะนำสำหรับการสร้างแฟ้ม .xml การตั้งค่าคอนฟิกสำหรับเครื่องมือการปรับใช้ Office ๒๐๑๖ 
- เชื่อมต่อไคลเอ็นต์เดสก์ท็อปของโครงการออนไลน์กับโครงการแบบออนไลน์

## <a name="sharepoint-online"></a>SharePoint Online

ให้คำแนะนำเกี่ยวกับ:
- การตั้งค่าการจัดเตรียมผู้ใช้ซึ่งรวมถึงการให้สิทธิ์
- การเปิดใช้งานการสร้างไซต์สำหรับผู้ดูแลระบบ SharePoint แบบออนไลน์ของคุณ    
- การวางแผนไซต์คอลเลกชัน 
- การรักษาความปลอดภัยเนื้อหาและการจัดการสิทธิ์
- การเปิดใช้งานเว็บไซต์ส่วนบุคคลและคุณลักษณะทางสังคม
- การกำหนดค่าลักษณะการทำงานแบบออนไลน์ของ SharePoint 
- ให้การย้ายข้อมูลถ้ามีใช้บริการการย้ายข้อมูลศูนย์ FastTrack และถ้าคุณพบสิทธิ์
- การประเมินการตั้งค่าคอนฟิกโครงสร้างพื้นฐานฟาร์ม SharePoint ในสถานที่ที่จำเป็นสำหรับไฮบริดสลี SharePoint แบบออนไลน์ 
- การใช้เครื่องมือและการทำงานอัตโนมัติเพื่อ:
  - กำหนดค่าโปรแกรมประยุกต์บริการการค้นหาระบบคลาวด์ในสถานที่ 
  - กำหนดค่าความน่าเชื่อถือระหว่างสภาพแวดล้อมของ SharePoint ในสถานและระบบคลาวด์
- การกำหนดค่าไซต์ SharePoint ในสถานที่ที่จะใช้คุณลักษณะไฮบริดสลี SharePoint แบบออนไลน์
    
## <a name="skype-for-business-online"></a>Skype สำหรับธุรกิจออนไลน์

ให้คำแนะนำเกี่ยวกับ:
- การเตรียมใช้งาน Skype สำหรับข้อมูลเฉพาะตัวของธุรกิจสำหรับ Office ๓๖๕ 
- การเปิดใช้งานการประชุมออนไลน์การส่งข้อความโต้ตอบแบบทันที (IM) และคุณลักษณะสถานะการออนไลน์สำหรับ Office ๓๖๕ 
- การสร้างบัญชีที่จะเชื่อมโยงกับอุปกรณ์ระบบห้องที่ได้รับการสนับสนุน (สูงสุด10บัญชี) 
- การตั้งค่าคอนฟิกสภาพแวดล้อมของเซิร์ฟเวอร์โดเมนที่แยกการสนับสนุน Lync ไฮบริดสลีหรือ Skype สำหรับสถานการณ์ไฮบริดสลีธุรกิจออนไลน์ (ถ้าเกี่ยวข้อง)
- การเปิดใช้งานการประชุมทางเสียง:
  - การตั้งค่าองค์กรสำหรับการประชุมที่เป็นค่าเริ่มต้นของบริดจ์
  - การกำหนดบริดจ์การประชุมให้กับผู้ใช้ที่ได้รับอนุญาต
- การเปิดใช้งานระบบโทรศัพท์:
  - การเปิดใช้งานระบบโทรศัพท์และแผนการโทรปฐมนิเทศ (ในตลาดที่มีอยู่)
  - การกำหนดหมายเลขให้กับผู้ใช้ที่ได้รับอนุญาต
  - หมายเลขท้องถิ่น porting คำแนะนำผ่าน UI ถึง๙๙๙
  - การสนับสนุนหมายเลขท้องถิ่น porting SR มากกว่า๙๙๙
- การเปิดใช้งาน Skype สำหรับการออกอากาศการประชุมทางธุรกิจ:
  - การเปิดใช้งาน Skype สำหรับการประชุมทางธุรกิจการออกอากาศปฐมนิเทศคำแนะนำ
  - การตั้งค่าองค์กรสำหรับสหพันธรัฐด้วยบริการการออกอากาศการประชุม
